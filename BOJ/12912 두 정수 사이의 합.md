## 아이디어
Implement  
  
`a`, `b` 제한이 커서 연속합 공식을 써야할 줄 알았는데  
안 써도 통과 된다  
  
반복문 조건을 `a<=b`로 할꺼라  
`a>b` 일 때 순서를 바꿔준다
```c
void swap(int *a, int *b) {
    int temp=*a;
    *a=*b;
    *b=temp;
}

if(a>b)
    swap(&a, &b);
```
`a`부터 `b`까지 모든 수를  
`answer`에 더한다
```c
long long answer = 0;
for( ; a<=b ; a++)
    answer+=a;
return answer;
```
