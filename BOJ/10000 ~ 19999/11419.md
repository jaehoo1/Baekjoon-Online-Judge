# [11419 : Olympic Parade](https://www.acmicpc.net/problem/11419)
https://www.acmicpc.net/problem/11419

## 문제
**문제**  
SEERC 조직위원회는 올해 개막식을 색다른 방식으로 진행하기로 결정했다. 참가자들의 도심 퍼레이드를 조직하는 것이다. 올해는 참가국의 대학을 대표하는 N명(참가자, 코치, 손님)이 거리로 나와 독창적인 의상과 큰 노래 소리로 관중을 사로잡으려 한다. 각 대학은 하나의 그룹으로 대표되며, 그룹에 속한 모든 사람은 해당 대학을 나타내는 고유 식별자(ID)를 가지고 있다. 퍼레이드를 잘 정돈되고 재미있게 만들기 위해, 각 그룹은 한 줄에 정확히 K명씩 여러 줄로 서야 한다.

하지만 단 하나의 대학만 이 규칙에 맞게 줄을 설 수 없었고, 당신은 그 대학의 ID를 찾아야 한다.

<br/>

**입력**  
입력의 첫 줄에는 두 정수 N과 K가 공백으로 구분되어 주어진다. (1 ≤ N ≤ 1,000,000, 2 ≤ K ≤ N) 이후 N개의 줄에 걸쳐 N명의 사람들의 ID C₁, C₂, ..., Cₙ이 주어진다. (0 ≤ Cᵢ ≤ 1,000,000,000, 1 ≤ i ≤ N)

<br/>

**출력**  
출력은 한 줄로, 문제의 정답인 하나의 정수(ID)를 출력한다.

<br/>

## 아이디어
(Hash) Map(Data Structure)

고유 식별자(C)가 나온 횟수를 `Map<Integer, Integer>`에 카운팅 해주고 `K`로 나눠떨어지지 않는 key를 찾는다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

Map<Integer, Integer> map = new HashMap<>();
StringTokenizer st = new StringTokenizer(br.readLine());
int N = Integer.parseInt(st.nextToken());
int K = Integer.parseInt(st.nextToken());
while (N-- > 0) {
    map.compute(Integer.parseInt(br.readLine()), (k, v) -> v == null ? 1 : v + 1);
}
System.out.println(map.entrySet().stream().filter(entry -> entry.getValue() % K != 0).findFirst().get().getKey());
```
