# [19796 : Реклама на заборе](https://www.acmicpc.net/problem/19796)
https://www.acmicpc.net/problem/19796

## 문제
**문제**  
이반은 작은 마을의 예쁜 집에 살고 있습니다. 그의 집 주변에는 최근에 빨간색으로 칠해진 울타리가 있습니다. 그러나 광고 전단지를 붙이고 다니는 광고 대행사원의 형태로 문명이 이반의 마을에 찾아왔고, 그의 울타리도 같은 운명을 맞이했습니다.

매일 그의 울타리에 새로운 광고가 붙여집니다. 이렇게 지난 $n$일 동안 울타리에는 이미 $n$개의 광고가 붙었고, 이반은 $m$개의 판자로 구성된 울타리 전체가 광고로 덮여있다고 생각합니다. 판자들은 울타리를 따라 1부터 $m$까지 번호가 매겨져 있습니다.

매일 광고 대행사원이 와서 광고를 붙일 때마다, 이반의 이웃인 표트르는 어떤 판자들이 이 광고로 덮였는지 기록했습니다. 구체적으로, $i$번째 날에 새 광고는 $l_i$번 판자부터 $r_i$번 판자까지 포함하여 붙여졌다는 것을 알게 되었습니다. 이때 광고 대행사원은 자신의 이전 광고를 전체적으로 또는 부분적으로 새 광고로 덮을 수 있었습니다.

마을 행정부에 민원을 제기하기 위해 이반은 울타리 전체가 광고로 덮여 있는지 확인해야 합니다. 이것이 사실인지 알아내는 것을 도와주세요.

<br/>

**입력**  
입력 파일의 첫 번째 줄에는 두 자연수 $m$과 $n$이 주어집니다 --- 울타리의 판자 수와 표트르가 관찰한 날짜 수입니다 ($1 \le m \le 10\,000$, $1 \le n \le 1000$). 이어서 $n$개의 줄에 정수 $l_i$, $r_i$가 주어집니다 ($1 \le l_i \le r_i \le m$). $i$번째 숫자 쌍은 $i$번째 날에 광고가 붙여진 울타리 구간을 설명합니다.

<br/>

**출력**  
울타리 전체가 광고로 덮여 있다면 "YES"를, 그렇지 않다면 "NO"를 출력하세요.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/152030

## 아이디어
Simulation(Implementation)

$n$번 울타리를 범위($l_i$, $r_i$)만큼 칠하며 $m$개의 울타리가 전부 덮여졌는지 확인
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

StringTokenizer st = new StringTokenizer(br.readLine());
int m = Integer.parseInt(st.nextToken());
int n = Integer.parseInt(st.nextToken());
boolean[] fences = new boolean[m + 1];
while (n-- > 0) {
    st = new StringTokenizer(br.readLine());
    int l = Integer.parseInt(st.nextToken());
    int r = Integer.parseInt(st.nextToken());
    for (int i = l; i <= r; i++) {
        fences[i] = true;
    }
}
boolean allCovered = true;
for (int i = 1; i <= m; i++) {
    if (!fences[i]) {
        allCovered = false;
    }
}
System.out.println(allCovered ? "YES" : "NO");
```
