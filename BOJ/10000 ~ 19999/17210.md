## 아이디어
Math, Implementation

N이 크다고 해도 전체 반복하기 전에 애초에 방을 탈출할 수가 없다.
```python
doors = [-1]        # 첫 방은 의미 없으니 인덱스 편의를 위해 빈 값
is_valid = True
N = int(input())
doors.append(int(input()))  # 처음 여는 방법
for i in range(2, N + 1):
    next = 0 if doors[i - 1] else 1     # 다음문은 이전과 반대로 열어야 함
    if (i > 2 and i % 2 == 0 and next != doors[2]) or (i > 3 and i % 3 == 0 and next != doors[3]):  # 2의배수 / 3의배수 번째는 2 / 3 인덱스와 같은 방법으로 열어야 함
        is_valid = False
        break
    doors.append(next)
print("\n".join(map(str, doors[2:])) if is_valid else "Love is open door")
```
