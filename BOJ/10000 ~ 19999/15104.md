# [15104 : Odd Palindrome](https://www.acmicpc.net/problem/15104)
https://www.acmicpc.net/problem/15104

## 문제
주어지는 문자열의 모든 부분 문자열의 팰린드롬들이 홀수 길이인지 확인하는 문제

## 아이디어
String

한 글자 한 글자는 각각 홀수 팰린드롬들이다.
```text
abc → a (홀수 팰린드롬), b (홀수 팰린드롬), c (홀수 팰린드롬)
```

붙어있는 같은 문자가 있으면 짝수 팰린드롬이게 된다.
```text
aaa → a (홀수 팰린드롬), a (홀수 팰린드롬), a (홀수 팰린드롬), aa (짝수 팰린드롬), aa (짝수 팰린드롬), aaa (홀수 팰린드롬)
```

즉, 붙어있는 문자가 같으면 짝수 팰린드롬이 있을 수 밖에 없다.
```java
static boolean containsEvenPalindrome(String s) {

    for (int i = 0; i < s.length() - 1; i++) {
        if (s.charAt(i) == s.charAt(i + 1)) {
            return true;
        }
    }
    return false;
}


System.out.println(containsEvenPalindrome(new Scanner(System.in).nextLine()) ? "Or not." : "Odd.");
```
