## 아이디어
Dynamic Programming  
가장 긴 증가하는 부분 수열의 길이를 구하는 문제  
  
N번째 원소 까지의 가장 긴 증가하는 부분 수열의 길이는  
N-1번째 원소 까지의 가장 긴 증가하는 부분 수열의 길이 중  
최댓값이면서 자신보다 원소값이 작은 값 + 1로 정의할 수 있다  
  
dp[i]는 dp[0]~dp[i-1] 값 중 최대값 & 자신보다 작은 원소값을 가진 원소 + 1이다
```c
for(i=1 ; i<N ; i++) {
	max=0;
	for(j=0 ; j<i ; j++)
		if(dp[j]>max&&arr[i]>arr[j])
			max=dp[j];
	dp[i]=max+1;
}
```
