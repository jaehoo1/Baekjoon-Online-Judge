# [19963 : Санта Клаус](https://www.acmicpc.net/problem/19963)
https://www.acmicpc.net/problem/19963

## 문제
**문제**  
메리 크리스마스! 산타 할아버지는 올해도 n명의 아이들에게 선물을 나눠 주기 위해, 크리스마스를 준비하고 있습니다. 도우미 엘프들은 두 개의 선물꾸러미에 선물을 나누어 담았고, 산타 할아버지가 헷갈리는 일이 없도록 꾸러미에 선물이 들어있는 아이들을 1번부터 n번까지로 번호로 구분한 명단을 만들었습니다.

산타 할아버지는 엘프들을 돕고자 아직 선물꾸러미에 담지 못한 선물들을 미리 세 번째 꾸러미에 모두 담아 놓기로 결정했습니다. 산타 할아버지를 도와 세 번째 꾸러미에 담아야 하는 아이들의 명단을 만들어 봅시다.

<br/>

**입력**  
첫째 줄에 아이들의 총 인원 수 $n$, 첫 번째 꾸러미에 있는 선물 수 $m$, 두 번째 꾸러미에 있는 선물 수 $k$가 주어집니다. 둘째 줄에 첫 번째 꾸러미에 선물이 들어 있는 아이들의 번호가 주어집니다. 셋째 줄에 두 번째 꾸러미에 선물이 들어 있는 아이들의 번호가 주어집니다.

<br/>

**출력**  
첫째 줄에 세 번째 꾸러미에 담아야 할 선물의 개수를 출력합니다. 둘째 줄에 세 번째 꾸러미에 선물이 들어 있어야 할 아이들의 번호를 출력합니다. 아이들의 명단을 출력할 때는 순서는 상관없습니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/101946

## 아이디어
Implementation, Set

`Set<Integer>`을 사용해 전체 받아야 하는 아이들 집합을 초기화하고 첫 번째 꾸러미, 두 번째 꾸러미의 아이들을 제거하면 `Set<>`에 남아있는 값들이 세 번째 꾸러미가 된다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

StringTokenizer st = new StringTokenizer(br.readLine());
int n = Integer.parseInt(st.nextToken());
int m = Integer.parseInt(st.nextToken());
int k = Integer.parseInt(st.nextToken());
// 전체 아이들 집합 초기화
Set<Integer> set = new HashSet<>();
for (int i = 1; i <= n; i++) {
    set.add(i);
}
// 첫 번째 꾸러미 제거
st = new StringTokenizer(br.readLine());
while (m-- > 0) {
    set.remove(Integer.parseInt(st.nextToken()));
}
// 두 번째 꾸러미 제거
st = new StringTokenizer(br.readLine());
while (k-- > 0) {
    set.remove(Integer.parseInt(st.nextToken()));
}

// set에 남아있는 원소가 세 번째 꾸러미
bw.write(set.size() + "\n");
bw.write(
        set.stream()
                .map(String::valueOf)
                .collect(Collectors.joining(" "))
);
```
