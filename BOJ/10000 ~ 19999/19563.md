## 아이디어
Parity, Math

좌표의 (유클리드) 거리는 $c$보다 같거나 작아야 갈 수가 있다
```python
abs(a) + abs(b) <= c
```

왔다 갔다 (제자리 뛰기(?))를 하면 거리만 낭비를 시킬 수가 있는데, `abs(a) + abs(b)`의 홀/짝과 `c`의 홀/짝이 같음으로 확인을 하면 된다.
```python
(abs(a) + abs(b)) % 2 == c % 2
```
```python
a, b, c = map(int, input().split())
print("YES" if abs(a) + abs(b) <= c and (abs(a) + abs(b)) % 2 == c % 2 else "NO")
```
