## 문제
N % X 를 출력하는 문제

## 아이디어
Big Integer(Arbitrary Precision), Arithmetic(Math)

단순히 N % X를 구하면 되기는 하는데, N의 범위가 굉장히 커서 큰 수 연산을 해주어야 한다.
> `python`의 `int`형은 기본적으로 제한 범위가 크므로 쉽게 구현이 가능하다.
```python
T = int(input())
for n in range(1, T + 1):
    N, X = map(int, input().split())
    print(f"Case {n}: {N % X}")
```
