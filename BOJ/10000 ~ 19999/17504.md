분수를 나타내는 클래스를 구현한다. 1 / 분수 연산이 가능하게 끔 분자/분모 를 뒤집는 기능도 구현한다.  
정수 + 분수 연산도 구현한다.
```java
class Fraction {

    long P, Q;

    public Fraction(long p, long q) {
        P = p;
        Q = q;
    }

    void reverse() {
        long temp = P;
        P = Q;
        Q = temp;
    }
}

static Fraction plus(int num, Fraction fraction) {
    return new Fraction(fraction.Q * num + fraction.P, fraction.Q);
}
```

처음 분수를 초기화하고 반복문을 통해 최종 분수를 구하고 1에서 뺀 값을 구한다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int N = Integer.parseInt(br.readLine());
int[] arr = new int[N];
StringTokenizer st = new StringTokenizer(br.readLine());
for (int i = 0; i < N; i++) {
    arr[i] = Integer.parseInt(st.nextToken());
}
Fraction fraction = new Fraction(1, arr[N - 1]);    // 처음 분수는 1 / 마지막정수
for (int i = N - 2; i >= 0; i--) {
    fraction = plus(arr[i], fraction);              // 정수 더하기
    fraction.reverse();                             // 1 / 분수 뒤집기
}
System.out.println(fraction.Q - fraction.P + " " + fraction.Q);     // 1 - 분수
```
