# [16391 : Zebras and Ocelots](https://www.acmicpc.net/problem/16391)
https://www.acmicpc.net/problem/16391

## 문제
**문제**  
유명한 시티 센트럴 동물원에는 얼룩말과 오셀롯 두 마리만 서식하고 있습니다. 이 마법 생물들은 하나의 키 큰 기둥에 모여 각각 아래 생물의 등 위에 서 있다. 동물원 종이 울릴 때마다, 더미에서 가장 낮은 오셀롯이 얼룩말로 변하고, 그 아래에 있는 모든 얼룩말(있다면)이 동시에 오셀롯으로 변합니다. 종이 울릴 때 더미에 오셀롯이 없으면 아무 일도 일어나지 않습니다. 사육사는 수년간 이 흥미로운 과정을 지켜봐 왔습니다. 오늘, 갑자기 그는 이 상황이 얼마나 더 계속될 수 있을지 궁금해지기 시작한다. 즉, 얼룩말과/또는 오셀롯 무더기가 있다면, 오셀롯이 사라지기 전까지 종이 울리는 횟수는 몇 번이나 될까요?

<br/>

**입력**  
입력은 1부터 60까지의 숫자 N과 N개의 줄로 구성되며, 각 선은 Z(얼룩말)나 O(오셀롯)의 문자로 구성됩니다. 이 문서들은 생물의 순서를 위(먼저)에서 아래(마지막)로 나타냅니다.

<br/>

**출력**  
출력은 종이 울려야 하는 횟수를 나타내는 한 정수여야 하며, 이는 더 이상 오셀롯이 없도록 하는 횟수입니다.

## 아이디어
Math

맨 처음에는 naive하게 구현했고, 당연히(?) 시간초과가 났다. $O(2^N)$
```java
static char[] creatures;
static int N;

static boolean contains(char creature) {

    for (char c : creatures) {
        if (c == creature) {
            return true;
        }
    }
    return false;
}

static void bell() {

    int lowestOcelotIndex;
    for (lowestOcelotIndex = N - 1; lowestOcelotIndex >= 0 && creatures[lowestOcelotIndex] != 'O'; lowestOcelotIndex--);
    creatures[lowestOcelotIndex] = 'Z';
    for (int i = lowestOcelotIndex + 1; i < N; i++) {
        if (creatures[i] == 'Z') {
            creatures[i] = 'O';
        }
    }
}


BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

N = Integer.parseInt(br.readLine());
creatures = new char[N];
for (int i = 0; i < N; i++) {
    creatures[i] = br.readLine().charAt(0);
}
int count;
for (count = 0; contains('O'); bell(), count++);
System.out.println(count);
```

N 범위를 보고 힌트를 얻었다. ≤ 60 → 이거 뭔가 2진수 문제 같은데?
- 예제 입력1 "ZOZ" → 101
  - 모든 비트가 1이 되려면 (2진수)10 필요 → (10진수) 2 = 답
- 예제 입력2 "OZZO" → 0110
  - 모든 비트가 1이 되려면 (2진수)1001 필요 → (10진수) 9 = 답

즉, Z를 1로 보고 O를 0으로 보는 이진수에서 모든 비트를 1로 만드는 문제
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int N = Integer.parseInt(br.readLine());
StringBuilder sb = new StringBuilder();
for (int i = 0; i < N; i++) {
    sb.append(br.readLine());
}
String s = sb.toString();
long result = 0;
for (int i = 0; i < N; i++) {
    if (s.charAt(N - i - 1) == 'O') {
        result |= (1L << i);
    }
}
System.out.println(result);
```
