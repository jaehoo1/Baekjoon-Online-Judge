# [17884 : Lost Lineup](https://www.acmicpc.net/problem/17884)
https://www.acmicpc.net/problem/17884

## 문제
Jimmy(1번)와 n번 친구들 사이에 있는 사람 수가 주어질 때 서있는 순서를 구하는 문제

## 아이디어
Implementation

있는 사람들 만큼이 추가해줘야 할 (배열) 인덱스가 된다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int n = Integer.parseInt(br.readLine());
int[] arr = new int[n];
arr[0] = 1;     // Jimmy
StringTokenizer st = new StringTokenizer(br.readLine());
for (int i = 1; i < n; i++) {
    // Jimmy 사이에 있는 사람 수 → 인덱스 추가
    arr[Integer.parseInt(st.nextToken()) + 1] = i + 1;
}
for (int i = 0; i < n; i++) {
    bw.write(arr[i] + " ");
}
```
