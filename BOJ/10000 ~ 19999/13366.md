# [13366 : Math Contest](https://www.acmicpc.net/problem/13366)
https://www.acmicpc.net/problem/13366

## 문제
**문제**  
입력되는 수 9의 배수인지 판별하는 문제

## 아이디어
Big Integer(Arbitrary Precision), Number Theory, Arithmetic(Math)

입력 범위가 매우 크기 때문에 배수 판별을 그대로 `%`로 하면 안된다.  
문자열로 입력 받고 각 자리 합을 구하면 9의 배수 판별할 수 있디.
```java
static boolean isMultipleOf9(String s) {
    int sum = 0;
    for (char c : s.toCharArray()) {
        sum += c - '0';
    }
    return sum % 9 == 0;
}


BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int T = Integer.parseInt(br.readLine());
while (T-- > 0) {
    bw.write(isMultipleOf9(br.readLine()) ? "YES\n" : "NO\n");
}
```
