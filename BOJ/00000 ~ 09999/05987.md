# [5987 : String Function Encoding](https://www.acmicpc.net/problem/5987)
https://www.acmicpc.net/problem/5987

## 아이디어
String, Implementation

문제의 function을 그대로 구현
```python
def function(N: int, S: str) -> str:
    return S[N:] + S
```

이것을 입력 문자열에 N번 반복하면 된다.
```python
Z = int(input())
for _ in range(Z):
    N, C, S = input().split()
    N = int(N)
    C = int(C)
    for _ in range(C):
        S = function(N, S)
    print(S)
```
