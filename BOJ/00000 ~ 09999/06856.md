# [6856 : Roll the Dice](https://www.acmicpc.net/problem/6856)
https://www.acmicpc.net/problem/6856

## 문제
m, n 까지의 눈이 있는 주사위 두 개를 던질 때 10이 나오는 경우의 수를 구하는 문제

## 아이디어
Brute Force

m, n 크기가 작아 모든 경우의 수를 (일일이 반복문으로) 구해봐도 된다.  
경우의 수가 1일 때 주의
```java
Scanner sc = new Scanner(System.in);
int m = sc.nextInt();
int n = sc.nextInt();
int ways = 0;
for (int i = 1; i <= m; i++) {
    for (int j = 1; j <= n; j++) {
        if (i + j == 10) {
            ways++;
        }
    }
}
if (ways == 1) {
    System.out.println("There is 1 way to get the sum 10.");
} else {
    System.out.println("There are " + ways + " ways to get the sum 10.");
}
```
