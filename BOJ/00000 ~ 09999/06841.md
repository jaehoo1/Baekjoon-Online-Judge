## 문제
**문제**  
주어지는 문자열이 Short Form에 해당하면 Translation을 출력하고, 해당하지 않으면 그 문자열 그대로 출력하는 문제

## 아이디어
String, Implementation

Short Form을 key:value 형태(`Map<String, String>` / Dictionary / ...)로 저장해놓고 꺼내어 쓴다.
```python
TRANSLATIONS = {
    "CU": "see you",
    ":-)": "I’m happy",
    ":-(": "I’m unhappy",
    ";-)": "wink",
    ":-P": "stick out my tongue",
    "(~.~)": "sleepy",
    "TA": "totally awesome",
    "CCC": "Canadian Computing Competition",
    "CUZ": "because",
    "TY": "thank-you",
    "YW": "you’re welcome",
    "TTYL": "talk to you later"
}

while True:
    line = input()
    print(TRANSLATIONS.get(line, line))
    if line == "TTYL":
        break
```
