## 문제
BMI를 구해 지수에 따른 비만 정도를 측정하는 문제

## 아이디어
Arithmetic(Math)

문제에서 알려주는 BMI 식을 통해 계산하고 조건 분기 구현
```python
weight = float(input())
height = float(input())
bmi = weight / (height * height)
if bmi > 25:
    print("Overweight")
elif bmi >=18.5:
    print("Normal weight")
else:
    print("Underweight")
```
