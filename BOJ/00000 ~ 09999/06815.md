## 문제
**문제**  
a와 b 사이의 수 중 제곱수 이면서 세제곱수인 수의 개수 구하기

## 아이디어
Math

Naive 하게 구현했더니 너무 느리다
```python
def is_square(num: int) -> bool:
    return math.isqrt(num) ** 2 == num

def is_cube(num: int) -> bool:
    return round(num ** (1/3)) ** 3 == num

count = 0
for i in range(a, b + 1):
    if is_square(i) and is_cube(i):
        count += 1
```

제곱수 이면서 세제곱수 이려면 ^6이면 된다.
```python
a = int(input())
b = int(input())
count = 0
i = 1
while True:
    cur = i ** 6
    if cur > b:
        break
    i += 1
    if cur >= a:
        count += 1
print(count)
```
