# [3787 : Count on Canton](https://www.acmicpc.net/problem/3787)
https://www.acmicpc.net/problem/3787

## 문제
(칸토어의) 유리수 열거법을 통해 n번째 항 찾기

## 아이디어
Implementation, Math

의외로 반복문으로 풀리는 문제였다. 지렁이 형태로 유리수가 늘어나니, 이에 맞춰 n번째 항을 구하는 메서드를 구현
```java
static String getPosition(int term) {

    int i = 1, j = 1, rows = 1, iter = 1;
    while (iter < term) {
        if (i == 1 && rows % 2 == 1) {
            j++;
            rows++;
        } else if (j == 1 && rows % 2 == 0) {
            i++;
            rows++;
        } else {
            if (rows % 2 == 1) {
                j++;
                i--;
            } else {
                i++;
                j--;
            }
        }
        iter++;
    }
    return i + "/" + j;
}
```

위의 메서드를 통해 입력 되는 n번째 항을 찾을 수 있다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

String line;
while ((line = br.readLine()) != null) {
    int num = Integer.parseInt(line);
    bw.write("TERM " + num + " IS " + getPosition(num) + "\n");
}
```
