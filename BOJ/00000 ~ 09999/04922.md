## 문제
![image](https://user-images.githubusercontent.com/44667299/147430816-aa793922-97f9-40fd-9188-02ffbf52be21.png)  
그림과 같은 규칙으로 번호를 쓸 때,  
N×N 칸의 오른쪽 모서리의 숫자를 구하라
## 아이디어
Dynamic Programming  
  
직접 5×5 칸 까지 구해보면  
|N|1|2|3|4|5|
|---|---|---|---|---|---|
|res|1|3|7|13|21|

임을 알 수 있다  
1→2 로 갈 때 2증가,  
2→3 로 갈 때 4증가,  
3→4 로 갈 때 6증가,  
4→5 로 갈 때 8증가, ...  
규칙을 적용하면 되겠다
```c
int i, dp[1000]={1}, temp=2;

for(i=1 ; i<1000 ; i++) {
	dp[i]=dp[i-1]+temp;
	temp+=2;
}
```
