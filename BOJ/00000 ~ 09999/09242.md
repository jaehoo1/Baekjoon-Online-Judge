## 아이디어
Parsing(String), Implementation

각 점들의 위치가 올바른 숫자의 점(별표) 위치와 같은지 비교하면 올바른 숫자 형태인지 알 수 있다.

그러려면 각 숫자별 올바른 위치가 필요하다(노가다).
```java
// 숫자별 점의 위치 x + y 값을 사전 정의
static final List<Integer>[] NUMBER_POSITIONS = new List[]{
        List.of(0, 1, 2, 3, 5, 6, 8, 9, 11, 12, 13, 14),        // 0
        List.of(2, 5, 8, 11, 14),                               // 1
        List.of(0, 1, 2, 5, 6, 7, 8, 9, 12, 13, 14),            // 2
        List.of(0, 1, 2, 5, 6, 7, 8, 11, 12, 13, 14),           // 3
        List.of(0, 2, 3, 5, 6, 7, 8, 11, 14),                   // 4
        List.of(0, 1, 2, 3, 6, 7, 8, 11, 12, 13, 14),           // 5
        List.of(0, 1, 2, 3, 6, 7, 8, 9, 11, 12, 13, 14),        // 6
        List.of(0, 1, 2, 5, 8, 11, 14),                         // 7
        List.of(0, 1, 2, 3, 5, 6, 7, 8, 9, 11, 12, 13, 14),     // 8
        List.of(0, 1, 2, 3, 5, 6, 7, 8, 11, 12, 13, 14)         // 9
};
```

입력 처리
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

String[] strings = new String[5];
for (int i = 0; i < 5; i++) {
    strings[i] = br.readLine();
}
char[][] chars = new char[5][strings[0].length()];

    chars[i] = strings[i].toCharArray();

```

입력 받은 숫자들의 별표 좌표를 구할 수 있는 함수 구현
```java
/**
 * 문자열 별표 좌표 구하기
 *
 * @throws ArrayIndexOutOfBoundsException 열 반복문(j)에서 IndexOutOfBounds 가능성
 * @param chars 별표 문자열
 * @param y 시작 열
 * @return 별표 좌표
 */
static List<Integer> getPositions(char[][] chars, int y) {

    List<Integer> positions = new ArrayList<>();
    for (int i = 0; i < 5; i++) {
        for (int j = 0; j < 3; j++) {
            if (chars[i][j + y] == '*') {
                positions.add(i * 3 + j);
            }
        }
    }
    return positions;
}
```

이제 별표 좌표를 구할 수 있으므로 별표 좌표를 가지고 올바른 숫자 형태인지, 어떤 숫자인지, 전체 수는 자를 수 있는(폭탄 해체가 가능한) 수인지를 확인할 수 있다.
```java
static boolean isValidNumber(List<Integer> positions) {
    for (int i = 0; i < 10; i++) {
        if (positions.equals(NUMBER_POSITIONS[i])) {
            return true;
        }
    }
    return false;
}

static char getNumber(List<Integer> positions) {
    for (int i = 0; i < 10; i++) {
        if (positions.equals(NUMBER_POSITIONS[i])) {
            return (char) (i + '0');
        }
    }
    return 0;
}

static boolean isCutAble(String s) {

    return Integer.parseInt(s) % 6 == 0;
}
```

구현한 메서드 로직들로 폭탄 해체 가능 여부를 판단
```java
StringBuilder sb = new StringBuilder();
for (int i = 0; i < chars[0].length; i += 4) {
    List<Integer> positions = null;
    try {
        positions = getPositions(chars, i);
    } catch (ArrayIndexOutOfBoundsException e) {    // Index 초과는 잘못된 형태를 의미
        System.out.println("BOOM!!");
        System.exit(0);
    }
    if (!isValidNumber(positions)) {
        System.out.println("BOOM!!");
        System.exit(0);
    }
    sb.append(getNumber(positions));
}
System.out.println(isCutAble(sb.toString()) ? "BEER!!" : "BOOM!!");
```
