# [2057 : 팩토리얼 분해](https://www.acmicpc.net/problem/2057)
https://www.acmicpc.net/problem/2057

## 아이디어
Brute Force

1,000,000,000,000,000,000 까지의 팩토리얼을 구해보면 20개인데 2^20 은 별로 크지 않아서 완전탐색으로 풀이가 가능할 것 같다.
```java
static List<Long> factorials = new ArrayList<>();

static void initFactorials() {

    factorials.add(1L);
    factorials.add(1L);
    for (int i = 2; ; i++) {
        long next = factorials.get(factorials.size() - 1) * i;
        if (next > 1000000000000000000L) {
            break;
        }
        factorials.add(next);
    }
}
```

20개의 팩토리얼 수(0! ~ 19!)들에 대해 나올 수 있는 모든 합 경우를 구해가며 N을 만들 수 있는지 구한다.
```java
static long N;

static void recur(long sum, int depth) {

    if (N == 0) {
        System.out.println("NO");
        System.exit(0);
    }
    if (sum == N) {
        System.out.println("YES");
        System.exit(0);
    }
    if (sum > N || depth == factorials.size()) {
        return;
    }

    recur(sum, depth + 1);
    recur(sum + factorials.get(depth), depth + 1);
}


initFactorials();
N = new Scanner(System.in).nextLong();
recur(0, 0);
System.out.println("NO");
```
