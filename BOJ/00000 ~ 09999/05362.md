# [5362 : Garbled Message](https://www.acmicpc.net/problem/5362)
https://www.acmicpc.net/problem/5362

## 문제
문자열의 모든 "iiing" 을 "th"로 바꾸는 문제

## 아이디어
String, Implementation

[`String.contains()`의 성능이 좋지 않기 때문에](/BOJ/30000%20~%2039999/35262.md#java) 시간 초과가 날 것이라 예상했으나,  
입력 크기가 작은건지 별 문제 없이 `String.contains()`, `String.replace()` 반복 호출로도 통과가 되었다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

String line;
while ((line = br.readLine()) != null) {
    while (line.contains("iiing")) {
        line = line.replace("iiing", "th");
    }
    bw.write(line + "\n");
}
```
