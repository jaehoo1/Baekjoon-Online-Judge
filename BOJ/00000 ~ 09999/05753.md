## 문제
**문제**  
테스트 케이스 별로 D행 N열이 주어지면 모든 수가 1인 열이 있는지를 찾는 문제

## 아이디어
Implementation

문제 잘못 읽어서 1로 채워진 행이 있는 걸 찾는건 줄 알았는데, 열이였다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

while (true) {
    StringTokenizer st = new StringTokenizer(br.readLine());
    int N = Integer.parseInt(st.nextToken());
    int D = Integer.parseInt(st.nextToken());
    if (N == 0 && D == 0) {
        break;
    }
    int[][] attendees = new int[D][N];
    for (int i = 0; i < D; i++) {
        st = new StringTokenizer(br.readLine());
        for (int j = 0; j < N; j++) {
            attendees[i][j] = Integer.parseInt(st.nextToken());
        }
    }
    boolean isAttendedAll = false;
    for (int j = 0; j < N; j++) {
        for (int i = 0; i < D; i++) {
            if (attendees[i][j] == 0) {
                break;
            }
            if (i == D - 1) {
                isAttendedAll = true;
            }
        }
    }
    bw.write(isAttendedAll ? "yes\n" : "no\n");
}
```
