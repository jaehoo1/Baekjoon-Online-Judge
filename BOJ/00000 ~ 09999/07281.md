# [7281 : Internetas](https://www.acmicpc.net/problem/7281)
https://www.acmicpc.net/problem/7281

## 문제
**문제**  
Tom의 인터넷 연결 상태가 좋지 않고 가끔 끊어지기도 합니다. Tom은 인터넷이 다시 연결될 때까지 얼마나 기다려야 하는지 알고 싶어 수시로 인터넷에 연결되어 있는지 확인합니다.

Tom는 처음과 마지막엔 인터넷에 연결되어 있었다고 기억합니다.

Tom이 인터넷 없이 가장 오래 지낼 수 있었던 시간을 알아내도록 도와주세요.

<br/>

**입력**  
첫 번째 줄에는 Tom이 측정한 횟수인 양의 정수 N이 하나 들어 있습니다.

이어지는 N개의 행에는 각각 2개의 정수가 포함됩니다: Tom이 i번째 측정한 시간(Ti)과 측정 결과(Mi). 측정 시 인터넷에 연결되어 있었다면 Mi는 1이고, Tom이 그 시간에 인터넷에 연결되어 있지 않았다면 0입니다.

측정값은 Ti의 오름차순으로 표시됩니다.

<br/>

**출력**  
Tom이 인터넷에 접속할 수 없었던 가장 긴 시간을 하나의 정수로 출력합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/140647

## 아이디어
Implementation

Ti가 1이 아닌 시간대는 0으로 치고 시간을 더해도 된다(측정하지 않았다면 인터넷에 연결되지 않았다고 봐도 됨)
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int N = Integer.parseInt(br.readLine());
int lastInternet = 1;
int maxTime = 0;
while (N-- > 0) {
    StringTokenizer st = new StringTokenizer(br.readLine());
    int T = Integer.parseInt(st.nextToken());
    int M = Integer.parseInt(st.nextToken());
    if (M == 1) {
        maxTime = Math.max(maxTime, T - lastInternet);
        lastInternet = T;
    }
}
System.out.println(maxTime);
```
