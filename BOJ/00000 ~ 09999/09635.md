# [9635 : Balloons Colors](https://www.acmicpc.net/problem/9635)
https://www.acmicpc.net/problem/9635

## 문제
**문제**  
각 문제에 풍선 색깔을 정하는 것은 매년 해야 할 과제 중 하나이며, 때로는 까다롭기도 합니다.

몇몇 참가자들은 난이도에 따라 특정 문제의 색상을 선택한다는 것을 알게 되었습니다. 예를 들어, 가장 쉬운 문제는 빨간색이고 가장 어려운 문제는 검은색입니다.

이러한 가정이 사실이 되길 원하지 않기 때문에, 가장 쉬운 문제와 가장 어려운 문제에 제약을 추가하기로 결정했습니다.

N개의 문제가 있으며, 1부터 N까지 번호가 매겨져 있습니다. 가장 쉬운 문제는 문제 번호 1이고, 가장 어려운 문제는 문제 번호 N입니다. 또한 N개의 고유 색상이 있으므로, 단순화를 위해 각 색상에 1부터 N까지의 고유 숫자를 부여하겠습니다.

각 색상을 정확히 1개의 문제에 할당하고, 가장 쉬운 문제는 색 X를 받지 않고, 가장 어려운 문제는 색 Y를 받지 않게 합니다.

N, X, Y와 색상 할당이 주어졌을 때, 이 할당이 위의 조건을 만족하는지 여부를 확인하는 것이 과제입니다.

<br/>

**입력**  
귀하의 프로그램은 하나 이상의 테스트 케이스에서 테스트됩니다. 입력의 첫 줄은 단일 정수 T로, 테스트 케이스 수(1 ≤ T ≤ 100)를 나타냅니다. 테스트 케이스 다음으로, 각 테스트 케이스의 첫 줄은 3개의 정수로 구성되며, 이 중 하나는 N X Y (3 ≤ N ≤ 100) 공간과 (1 ≤ X , Y ≤ N)로 구분되며, 이는 문제의 수, 가장 쉬운 문제가 가져가지 말아야 할 색상, 그리고 가장 어려운 문제가 가져가지 말아야 할 색상을 나타냅니다. 각각 그 뒤에 N개의 정수가 한 칸으로 구분된 선이 따라오는데(1부터 N까지의 정수는 정확히 한 번씩 나타야 합니다), 첫 번째 정수가 첫 번째 문제(가장 쉬운 문제)의 색상, 두 번째 정수가 두 번째 문제의 색상, 이런 식으로 이어집니다(마지막 정수가 가장 어려운 문제의 색상입니다).

<br/>

**출력**  
각 테스트 케이스에 대해 한 줄을 출력하는데, 그 단어는 (따옴표 제외) 다음과 같습니다:
- "BOTH": 가장 쉬운 문제와 가장 어려운 문제 모두에 가져서는 안 되는 색이 나온다면.
- "EASY": 가장 쉬운 문제에도 색이 붙으면 안 되는 색이 붙는다면.
- "HARD": 가장 어려운 문제에도 색이 붙으면 좋겠는데, 그게 있으면 안 되는 색이 붙는다면.
- "OKAY": 가장 쉬운 문제와 가장 어려운 문제 모두에 색을 부여한다면, 그 색깔은 얻을 수 있습니다.

## 아이디어
Implementation

풍선 색들이 주어지면, 가장 앞이 X인가와 가장 뒤가 Y인가에 따라 출력을 분기처리 해주면 된다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int T = Integer.parseInt(br.readLine());
while (T-- > 0) {
    StringTokenizer st = new StringTokenizer(br.readLine());
    int N = Integer.parseInt(st.nextToken());
    int X = Integer.parseInt(st.nextToken());
    int Y = Integer.parseInt(st.nextToken());
    st = new StringTokenizer(br.readLine());
    int[] colors = new int[N];
    for (int i = 0; i < N; i++) {
        colors[i] = Integer.parseInt(st.nextToken());
    }

    // 가장 앞 풍선
    boolean easy = colors[0] == X;
    // 가장 뒷 풍선
    boolean hard = colors[N - 1] == Y;

    if (easy && hard) {
        bw.write("BOTH\n");
    } else if (easy) {
        bw.write("EASY\n");
    } else if (hard) {
        bw.write("HARD\n");
    } else {
        bw.write("OKAY\n");
    }
}
```
