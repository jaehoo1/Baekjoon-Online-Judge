# [7279 : Filling Shapes](https://www.acmicpc.net/problem/7279)
https://www.acmicpc.net/problem/7279

## 문제
**문제**  
버스 노선은 N개의 정류장으로 구성되어 있습니다. 버스는 공원에서 첫 번째 정류장까지 바로 오기 때문에 항상 비어있습니다. 그 이후부터는 각 정류장에서 승객이 승하차합니다.

승객들은 앉는 것을 좋아하기 때문에 버스에 자리가 있으면 앉고 자리가 없을 때만 서 있습니다. 버스에는 총 K 좌석이 있습니다.

각 정류장에서 몇 명의 승객이 승하차할지 알고 있습니다. 한 번에 버스에 서 있어야 하는 최대 승객 수를 구합니다.

<br/>

**입력**  
첫 번째 행에는 두 개의 정수 N과 K가 포함됩니다. N은 버스 노선의 정류장 수이고 K는 버스의 좌석 수입니다.

N의 다음 두 행에는 두 개의 숫자 ai와 bi가 포함됩니다. ai는 정류장에서 탑승하는 승객의 수이고 bi는 하차하는 승객의 수입니다.

<br/>

**출력**  
한 번에 버스에 서야 하는 최대 승객 수인 정수를 출력합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/140987

## 아이디어
Implementation

각 케이스 별 하차인원, 승차인원을 현재 버스에 탄 사람에 증감연산해 턴(정류장) 별 서있는 사람의 수를 구한다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
StringTokenizer st = new StringTokenizer(br.readLine());
int N = Integer.parseInt(st.nextToken());
int K = Integer.parseInt(st.nextToken());
int busPeople = 0;
int maxStanding = 0;
while (N-- > 0) {
    st = new StringTokenizer(br.readLine());
    int a = Integer.parseInt(st.nextToken());
    int b = Integer.parseInt(st.nextToken());
    busPeople += a - b;
    int standing = K > busPeople ? 0 : busPeople - K;
    maxStanding = Math.max(maxStanding, standing);
}
System.out.println(maxStanding);
```
