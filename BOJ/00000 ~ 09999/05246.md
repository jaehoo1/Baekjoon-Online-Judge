# [5246 : Checkerboard Rows](https://www.acmicpc.net/problem/5246)
https://www.acmicpc.net/problem/5246

## 문제
체스 말 위치가 주어질 때 행당 있을 수 있는 말의 최대 개수 구하기

## 아이디어
Implementation

좌표 저장 후에 행 개수가 같은 것 별로 필터링 해서 최대 개수를 구하면 된다.

체스 말 클래스 (좌표만 필요함)
```java
class Piece {

    int x, y;

    public Piece(int x, int y) {
        this.x = x;
        this.y = y;
    }
}
```

체스 말들을(좌표) 입력받는다. 체스보드는 (1,1) ~ (8,8) 까지니까 행은 1~8 까지다.  
각 행 별로 말 들이 몇개 있는지 구하고 그 중 최대값을 찾는다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int T = Integer.parseInt(br.readLine());
while (T-- > 0) {
    StringTokenizer st = new StringTokenizer(br.readLine());
    int N = Integer.parseInt(st.nextToken());
    Piece[] pieces = new Piece[N];
    for (int i = 0; i < N; i++) {
        int c = Integer.parseInt(st.nextToken());
        int r = Integer.parseInt(st.nextToken());
        pieces[i] = new Piece(r, c);
    }
    bw.write(IntStream.rangeClosed(1, 8)
            .mapToLong(i -> Arrays.stream(pieces).filter(piece -> piece.x == i).count())
            .max()
            .getAsLong() + "\n");
}
```
