# [3512 : Flat](https://www.acmicpc.net/problem/3512)
https://www.acmicpc.net/problem/3512

## 문제
당신은 부동산 업자를 위해 소프트웨어를 개발하는 개발자입니다. 당신은 부동산 업자가 중개하는 아파트에 대해 다양한 자료를 계산하는 기능을 구현해야 합니다. 각 아파트는 침실, 욕실, 주방, 발코니 등 다양한 유형의 방들로 이루어져 있습니다.

아파트의 비용은 계산면적과 단위면적당 비용을 곱한 값으로 계산됩니다. 계산면적은 발코니를 제외한 모든 방의 면적과 발코니의 면적의 절반의 합으로 계산됩니다.

아파트의 각 방의 면적과 단위면적당 비용이 입력으로 주어질 때 아래와 같은 값을 계산해야 합니다.
- 모든 방의 총 면적
- 침실의 총 면적
- 아파트의 비용

<br/>

**입력**  
입력의 첫 번째 줄에는 두 정수 n(1 ≤ n ≤ 10) 과 c(1 ≤ c ≤ 100,000)가 주어집니다. 각각 아파트의 방의 개수와 단위면적당 비용입니다.

이후 n개의 줄에 걸쳐 한 줄에 하나의 방에 대한 정보가 주어집니다. 각 줄에는 방의 면적을 나타내는 정수 ai(1 ≤ ai ≤ 100)와 방의 종류를 나타내는 단어 ti가 주어집니다. 방의 종류는 “bedroom”, “bathroom”, “kitchen”, “balcony”, “other” 중 하나입니다.

<br/>

**출력**  
첫 번째 줄에는 모든 방의 총 면적을 나타내는 정수를 출력해야 합니다. 두 번째 줄에는 "bedroom"의 총 면적을 나타내는 정수를 출력해야 합니다. 세 번째 줄에는 아파트의 비용을 나타내는 실수를 출력해야 합니다. 아파트의 비용은 실제 정답과 출력값의 절대오차 또는 상대오차가 10−6 이하여야 합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/120751

## 아이디어
Arithmetic(Math), String, Implementation

모든 방의 면적 합, 침실 면적 합, 발코니는 절반인 전체 면적 합을 구한다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

StringTokenizer st = new StringTokenizer(br.readLine());
int n = Integer.parseInt(st.nextToken());
int c = Integer.parseInt(st.nextToken());
int totalArea = 0;
int totalAreaOfBedrooms = 0;
double reducedTotalArea = 0;
while (n-- > 0) {
    st = new StringTokenizer(br.readLine());
    int a = Integer.parseInt(st.nextToken());
    String t = st.nextToken();
    totalArea += a;
    if ("bedroom".equals(t)) {
        totalAreaOfBedrooms += a;
    }
    reducedTotalArea += "balcony".equals(t) ? (double) a / 2 : a;
}
System.out.println(totalArea);
System.out.println(totalAreaOfBedrooms);
System.out.println(reducedTotalArea * c);
```
