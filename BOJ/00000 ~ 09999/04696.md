# [4696 : St. Ives](https://www.acmicpc.net/problem/4696)
https://www.acmicpc.net/problem/4696

## 문제
중세의 여행 상인 로버트 채프만은 옷이나 리본, 바늘을 팔기 위해 고향 마을과 세인트 아이브스를 정기적으로 여행하였습니다. 그러던 도중 그는 신기한 행렬과 마주하게 됩니다.
```
  세인트 아이브스로 여행을 갔을 때,
  아내가 일곱 명인 남자를 만났는데,
  아내는 각자 7개의 자루를 가졌고,
  각 자루마다 고양이가 7 마리 있었으며
  모든 고양이는 7마리의 아기 고양이를 데리고 있었습니다.
  얼마나 많은 고양이, 자루, 사람이 세인트 아이브스로 여행을 갔을까요 ?
```

사실, 이 오래된 수수께끼에 대한 답은 '하나'입니다. 세인트 아이브스로 여행을 간 것은 로버트 한 명 뿐이고, 로버트를 제외한 모두는 이미 세인트 아이브스에서 여행 중 이기 때문입니다.  하지만, 만약 얼마나 많은 고양이, 자루, 사람이 세인트 아이브스를 여행중인지 묻는다면 다음과 같이 계산할 수 있습니다.
- 1 명의 남자
- 7 명의 아내
- 7 * 7 개의 자루
- 7 * 7 * 7 마리의 고양이
- 7 * 7 * 7 * 7 마리의 아기 고양이

총 2801 입니다.

로버트는 다음 세인트 아이브스 여행에서 같은 남자를 만났고, 이번에는 3명의 아내를 동반했습니다. 3개의 자루에 3마리의 고양이 등 .. 으로 말입니다. 마을의 의식처럼 보이는 것이 궁금해진 로버트는  그 다음 해 동안 그와 마주칠 때마다 얼마나 많은 사람들이 그와 함께 여행했는지 기록했습니다. 로버트가 세인트 아이브스를 여행하는 동안 마주친 행렬의 크기는 각각 얼마입니까?

<br/>

**입력**  
입력은 여러 개의 데이터로 주어집니다. 각 데이터는 로버트가 그 해에 만난 아내, 아내당 자루, 고양이당 고양이의 수를 나타내는 실수가 주어지며 입력의 마지막에는 0이 입력됩니다.

<br/>

**출력**  
각 입력에 대해 행렬의 크기를 소수점 아래 2자리로 출력합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/128179

## 아이디어
Arithmetic(Math)

n^0 ~ n^4 를 더하면 된다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

while (true) {
    double num = Double.parseDouble(br.readLine());
    if (num == 0) {
        break;
    }
    bw.write(String.format("%.2f\n", IntStream.rangeClosed(0, 4)
            .mapToDouble(i -> Math.pow(num, i))
            .sum()));
}
```
