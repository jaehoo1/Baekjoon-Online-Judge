## 문제
매주 월요일 : X 쿠나  
매주 화요일 : X + K 쿠나  
매주 수요일 : X + 2×K 쿠나  
...  
매주 일요일 : X + 6×K 쿠나를 예금한다  
  
52주간 예금할 때 정확히 N쿠나를 만드려면 X와 K의 값을 구하여라  
(답이 여러개가 나올 수 있는 경우에는 X가 가장 큰 값을 출력하라)  

## 아이디어
364X + 1092K == N 이 되면 된다. X 범위가 100이 최대이므로 X를 100부터 1식 줄여나가면서  
가능한 값을 찾는다
```
int N, X=100, K;
scanf("%d", &N);
while(X) {
	K=(N-X*364)/1092;
	if((N-X*364)%1092==0&&K>0) {
		printf("%d\n%d\n", X, K);
		break;
	}
	X--;
}
```
