## 문제
주어지는 문자열 중 2개이상 붙어있는 'a' 개수를 구하는 문제

## 아이디어
Brute Force, String, Implementation

(OutOfIndex에 주의하며) 2개이상 붙어있는 'a' 개수를 구한다.
```python
N = int(input())
s = input()
count = 0
for i in range(N):
    if (i < N - 1 and s[i] == 'a' and s[i + 1] == 'a') or (i > 0 and s[i] == 'a' and s[i - 1] == 'a'):
        count += 1
print(count)
```
