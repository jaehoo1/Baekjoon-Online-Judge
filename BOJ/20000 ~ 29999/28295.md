## 아이디어
Simulation(Implementation)

방향 회전을 구현 (동서남북 4 방향이므로, 증감 연산 후 범위 초과 보정 / `%` 연산으로 방향을 확정할 수 있다.)
```python
dir = 0
for _ in range(10):
    t = int(input())
    if t == 1:
        dir += 1
    elif t == 2:
        dir += 2
    elif t == 3:
        dir -= 1

    # 방향 범위 보정
    if dir < 0:
        dir += 4
    elif dir >= 4:
        dir -= 4
print("NESW"[dir])
```
