# [26336 : Are We Stopping Again?](https://www.acmicpc.net/problem/26336)

## 문제
Orooji 박사님과 가족은 자동차 여행을 떠나려 합니다. 당연히 연료를 보급하기 위해 멈춰야 하지만, 아이들이 먹고 싶어 할 때마다 멈추기도 합니다. O 박사님은 여행을 떠나기 전에 멈춰야 할 횟수를 파악하여 정신적으로 대비해야 합니다.

O 박사님의 여행에서 총 멈추는 횟수를 다음 정보를 바탕으로 구하세요:
1. 총 이동할 거리 (마일).
2. 주유를 위해 얼마나 자주 멈추는지 (마일 단위).
3. 식사를 위해 얼마나 자주 멈추는지 (마일 단위).

차량의 연료 탱크는 여행 시작 시 가득 차 있고 아이들도 배부른 상태라고 가정합니다. 목적지가 주유(또는 식사) 시간과 겹친다면, 그곳은 정차 횟수에 포함하지 마세요. 또한, 특정 마일리지가 주유 시간과 식사 시간 모두에 해당한다면, 두 번이 아닌 한 번의 정차로 계산합니다.

특정 마일리지가 주유 시간만 해당한다면, 그 정차에서는 아이들이 식사하지 않습니다. 마찬가지로, 특정 마일리지가 식사 시간만 해당한다면, 그 정차에서는 차량에 연료를 보급하지 않습니다.

<br/>

**입력**  
첫 번째 입력 줄에는 확인할 여행의 수를 나타내는 양의 정수 t가 포함됩니다. 여행 정보는 다음 t개의 입력 줄에 한 줄당 하나의 여행 정보로 제공됩니다. 각 여행은 세 개의 정수(각각 1에서 1000 사이)를 제공합니다. 이들은 위에서 지정된 세 가지 값을 순서대로 나타냅니다.

<br/>

**출력**  
각 여행에 대해 세 개의 입력 값을 출력합니다. 그런 다음, 다음 출력 줄에 해당 여행의 정차 횟수를 출력합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/160615

## 아이디어
Euclidean, Number Theory, Arithmetic, Math

```python
import math

t = int(input())
for _ in range(t):
    miles, gas, food = map(int, input().split())
    # 밑의 뺄셈 연산을 하기 전 입력 값 출력
    print(miles, gas, food)
    # 목적지가 주유/식사 지점인 경우 제외
    miles -= 1
    # 주유 + 식사 - 주유&식사 횟수
    print(miles // gas + miles // food - miles // math.lcm(gas, food))
```
