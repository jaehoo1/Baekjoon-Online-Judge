# [23739 : 벼락치기](https://www.acmicpc.net/problem/23739)
https://www.acmicpc.net/problem/23739

## 아이디어
Simulation(Implementation)

과목별 제한시간은 30분이고, 30분 안에 절반 이상을 풀 수 있나를 확인하면 됨
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int N = Integer.parseInt(br.readLine());
int[] T = new int[N];
for (int i = 0; i < N; i++) {
    T[i] = Integer.parseInt(br.readLine());
}
int count = 0;
int time = 30;
for (int i = 0; i < N; i++) {
    if (time >= T[i]) {
        time -= T[i];
        count++;
    } else if (time >= (double) T[i] / 2) {     // 실수 처리 안하면 홀수 경우에 틀림
        time = 0;
        count++;
    } else {
        time = 0;
    }
    if (time == 0) {
        time = 30;
    }
}
System.out.println(count);
```
