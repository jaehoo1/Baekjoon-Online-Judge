# [29918 : Leiutaja number üks](https://www.acmicpc.net/problem/29918)
https://www.acmicpc.net/problem/29918

## 문제
발명가 마을에서 큰 경진이 열렸습니다! 10주 후에 가장 많은 발명품을 선보인 사람이 최우수 발명가 칭호를 받게 됩니다.

아달베르트가 경진대회 소식을 듣고는 즉시 오스카르, 조반니, 헬미, 바이노, 심지어 아베 박사까지 그들의 창문 뒤에서 몰래 지켜보며 그들이 이미 얼마나 많은 발명품을 준비해 두었는지 알아내려고 했습니다. 또한 그는 각자가 주당 몇 개의 발명품을 추가로 만들 수 있는지 알고 있습니다.

하지만 이제 그는 자신의 준비물과 발명 속도가 10주 후에 가장 많은 발명품을 선보이기에는 부족할 것 같다는 의심을 품고 있습니다. 그래서 그는 스스무를 통해 일본에서 몇 가지 이국적인 기기들을 주문해 대회에서 자신의 발명품으로 선보이려고 합니다.

아달베르트는 몇 개의 기기를 주문해야 자신의 발명품과 합쳐서 가장 많은 수를 가질 수 있을까요?

<br/>

**입력**  
입력은 정확히 6행으로, 각 행에는 공백으로 구분된 두 개의 정수가 있습니다. 첫 번째 행에는 $N$와 $M$이 입력으로 주어집니다.

이는 아달베르트가 현재 $N$개의 발명을 보유하고 있으며 매주 $M$개의 새로운 발명을 추가로 만들 수 있음을 의미합니다. 

다음 다섯 줄에는 아달베르트의 경쟁자들에 대한 동일한 데이터가 있습니다. 어떤 숫자도 0보다 작거나 100보다 크지 않습니다.

<br/>

**출력**  
아달베르트가 일본에서 몇 개의 기기를 추가로 주문해야 대회 당일에 

가장 많은 발명품을 선보일 수 있는지를 나타내는 정수 $K$를 출력합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/160487

## 아이디어
Arithmetic(Math), Implementation

$N$과 $M$이 주어지면 최종 만들 수 있는 발명품 개수는 $N$ + $10M$
```java
static int productCount(int N, int M) {
    return N + 10 * M;
}

static int productCount(String N, String M) {
    return productCount(Integer.parseInt(N), Integer.parseInt(M));
}
```

Adalbert의 최종 발명품 개수와 다른 5명의 최종 발명품 개수를 구해 몇 개를 더 만들어야 하는지 구한다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

StringTokenizer st = new StringTokenizer(br.readLine());
int adalbert = productCount(st.nextToken(), st.nextToken());
int max = 0;
for (int i = 0; i < 5; i++) {
    st = new StringTokenizer(br.readLine());
    max = Math.max(max, productCount(st.nextToken(), st.nextToken()));
}
System.out.println(max < adalbert ? 0 : max - adalbert + 1);
```
