# [21022 : Three Points for a Win](https://www.acmicpc.net/problem/21022)
https://www.acmicpc.net/problem/21022

## 문제
축구 경기 별 점수가 주어질 때, 최종 승패 점수를 구하는 문제

## 아이디어
Arithmetic(Math), Implementation

각 경기별 팀별 점수를 입력받고
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
int N = Integer.parseInt(br.readLine());
int[] scores = new int[N];
int[] opponent = new int[N];
StringTokenizer st = new StringTokenizer(br.readLine());
for (int i = 0; i < N; i++) {
    scores[i] = Integer.parseInt(st.nextToken());
}
st = new StringTokenizer(br.readLine());
for (int i = 0; i < N; i++) {
    opponent[i] = Integer.parseInt(st.nextToken());
}
```

경기별 승패에 따른 점수를 합한다.
```java
int score = 0;
for (int i = 0; i < N; i++) {
    if (scores[i] > opponent[i]) {          // 승
        score += 3;
    } else if (scores[i] == opponent[i]) {  // 무
        score++;
    }
    // 패는 0점
}
System.out.println(score);
```
