## 아이디어
Brute Force

$2^x + 2^y = m$이므로 ($2^n$ 이므로) $x$, $y$ 증가에 따른 결과값 증가폭이 매우 커 실제 반복을 해야하는 횟수가 적다. 따라서 모든 경우의 수를 탐색해 답을 찾으면 된다.
```python
n = int(input())
for _ in range(n):
    m = int(input())
    x = 0
    while 2 ** x < m:
        y = x
        while 2 ** x + 2 ** y <= m:
            if 2 ** x + 2 ** y == m:
                print(f"{x} {y}\n", end="")
            y += 1
        x += 1
```
