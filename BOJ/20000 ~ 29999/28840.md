# [28840 : Как покормить дракона](https://www.acmicpc.net/problem/28840)
https://www.acmicpc.net/problem/28840

## 문제
오늘의 시간, 내일의 시간이 주어지면 사이의 시간을 구하는 문제

## 아이디어
Arithmetic(Math), String

계산 편의성을 위해 모든 시간은 분으로 계산한다.
```java
Scanner sc = new Scanner(System.in);

// 오늘 시간
StringTokenizer st = new StringTokenizer(sc.nextLine(), ":");
int today = Integer.parseInt(st.nextToken()) * 60 + Integer.parseInt(st.nextToken());

// 내일 시간
st = new StringTokenizer(sc.nextLine(), ":");
int tomorrow = Integer.parseInt(st.nextToken()) * 60 + Integer.parseInt(st.nextToken()) + 60 * 24;

// 내일 시간 - 오늘 시간
tomorrow -= today;
System.out.printf("%02d:%02d", tomorrow / 60, tomorrow % 60);
```
