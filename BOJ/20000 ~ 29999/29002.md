# [29002 : Космический корабль](https://www.acmicpc.net/problem/29002)
https://www.acmicpc.net/problem/29002

## 문제
**문제**  
엔젤 그로브 마을을 구하는 과정에서 블랙 레인저 잭 테일러는 또 다른 장애물에 직면했습니다. 레인저는 적들에게 둘러싸인 채 외계인 우주선에 갇혔고, 이제 탈출하기 위해서는 특정한 순서로 모든 적을 제거해야 합니다.

 
$n$명의 적들 각각은 힘 $f_i$를 가지고 있습니다. 그러나 그들 중에는 다른 모든 적들의 힘의 합과 같은 힘을 가진 보스가 있습니다. 보스를 제거하는 것은 완전한 집중력을 요구하기 때문에, 잭은 다른 모든 적들을 제거한 후에만 보스를 처리할 수 있습니다.

레인저는 시간이 얼마 없어서 누가 보스인지 파악할 시간이 없습니다. 그에게 여러분의 도움이 필요합니다. 잭 테일러가 탈출하기 위해 적들을 제거해야 하는 순서를 찾아주세요.

<br/>

**입력**  
입력 파일의 첫 번째 줄에는 적의 수를 나타내는 자연수 $n$이 있습니다 ($3 \le n \le 10^5$).

두 번째 줄에는 각 적의 힘을 나타내는 $n$개의 정수 $f_i$가 있습니다 ($-10^9 \le f_i \le 10^9$). 적들의 힘은 무작위 순서로 주어집니다.

<br/>

**출력**  
출력 파일의 유일한 줄에 레인저가 적들을 제거할 순서대로 $f_i$ 값들을 출력하세요. 가능한 순서가 여러 개라면, 아무거나 출력하면 됩니다.

해답이 항상 존재하며, 보스가 될 수 있는 적이 정확히 한 명 있다는 것이 보장됩니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/152087

## 아이디어
Prefix Sum, Arithmetic(Math)

전체 합 - 적의 힘 == 적의 힘 인 적이 보스다.

미리 합을 구해놓고(prefix sum) 각 적이 보스인지 확인한다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int n = Integer.parseInt(br.readLine());
int[] forces = new int[n];
long sum = 0;
int bossIdx = -1;
StringTokenizer st = new StringTokenizer(br.readLine());
// 입력 받으면서 전체 적들 힘 합을 구해 놓음
for (int i = 0; i < n; i++) {
    forces[i] = Integer.parseInt(st.nextToken());
    sum += forces[i];
}
// 인덱스 순회하며 보스를 찿음
for (int i = 0; i < n; i++) {
    if (sum - forces[i] == forces[i]) {
        bossIdx = i;
    }
}
// 보스를 제외한 나머지 처치(출력)
for (int i = 0; i < n; i++) {
    if (bossIdx != i) {
        bw.write(forces[i] + " ");
    }
}
// 보스 처치(출력)
bw.write("" + forces[bossIdx]);
```
