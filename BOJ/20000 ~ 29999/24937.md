# [24937 : SciComLove (2022)](https://www.acmicpc.net/problem/24937)
https://www.acmicpc.net/problem/24937

## 아이디어
String

10번을 반복하면 문자열은 원상태가 된다. 즉, $N$을 10으로 나눈 나머지 만큼의 연산(문자열의 가장 첫 문자를 떼어낸 뒤, 문자열의 맨 뒤에 붙입니다.)만 하면 된다.
```java
Scanner sc = new Scanner(System.in);

String str = "SciComLove";
int N = sc.nextInt();
N %= str.length();
while (N-- > 0) {
    str = str.substring(1) + str.charAt(0);
}
System.out.println(str);
```
