# [26509 : Triangle](https://www.acmicpc.net/problem/26509)
https://www.acmicpc.net/problem/26509

## 문제
**문제**  
한 개의 선분으로 하나의 직사각형을 자르고 얻어진 조각들을 자유롭게 회전하고 뒤집어서 주어진 변 길이의 삼각형 두 개로 만들 수 있는지 여부를 결정합니다.

힌트: 피타고라스의 정리는 직각삼각형에서 a² + b² = c²의 관계가 성립한다고 말합니다. 여기서 c는 빗변의 길이이고 a와 b는 나머지 두 변의 길이입니다.

<br/>

**입력**  
입력의 첫 번째 줄에는 문제 세트의 개수를 나타내는 양의 정수 n이 주어집니다. 각 문제 세트는 두 줄로 구성됩니다.

첫 번째 줄에는 첫 번째 삼각형의 원하는 변의 길이를 나타내는 세 개의 양의 정수가 공백으로 구분되어 주어집니다.

마찬가지로, 두 번째 줄에는 두 번째 삼각형의 원하는 변의 길이를 나타내는 세 개의 양의 정수가 공백으로 구분되어 주어집니다.

변의 길이가 유효한 직각삼각형을 생성하지 않을 수도 있습니다. 모든 변의 길이는 100 이하입니다.

<br/>

**출력**  
각 문제에 대해 한 줄에 출력할 때 주어진 변 길이의 삼각형으로 만들 수 있도록 절단할 수 있는 직사각형이 있는지 여부를 나타냅니다. 이러한 직사각형이 있으면 YES를 출력합니다. 그렇지 않으면 NO를 출력합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/119711

## 아이디어
Pythagoras(Geometry, Arithmetic, Math), Sorting

삼각형 두 개를 합쳐 직사각형으로 만들 수 있는지를 판별하기

삼각형 두 개의 변이 주어지면 직각삼각형이고, 합쳐서 직사각형이 되는지를 판별하는 함수를 구현
```java
/**
 * 직각삼각형 두 개로 직사각형을 만들 수 있는지 판별
 *
 * @param triangle1 삼각형1의 변들
 * @param triangle2 삼각형2의 변들
 * @return 심각형1 + 삼각형2 == 직사각형이 유효한지
 */
static boolean isValidTraiangle(int[] triangle1, int[] triangle2) {
    // 변 순서대로 정렬 (변 길이 같음 비교를 위해)
    Arrays.sort(triangle1);
    Arrays.sort(triangle2);
    // 위에서 정렬했으니, 직사각형이 만들어지려면 각 삼각형 변 길이가 같아야함
    for (int i = 0; i < 3; i++) {
        if (triangle1[i] != triangle2[i]) {
            return false;
        }
    }
    // 삼각형이 직각삼각형인지 확인
    return triangle1[0] * triangle1[0] + triangle1[1] * triangle1[1] == triangle1[2] * triangle1[2];
}
```

삼각형 변들을 입력받고 합쳐서 직사각형이 만들어지는지 확인
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int n = Integer.parseInt(br.readLine());
while (n-- > 0) {
    int[] triangle1 = new int[3];
    int[] triangle2 = new int[3];
    StringTokenizer st = new StringTokenizer(br.readLine());
    for (int i = 0; i < 3; i++) {
        triangle2[i] = Integer.parseInt(st.nextToken());
    }
    st = new StringTokenizer(br.readLine());
    for (int i = 0; i < 3; i++) {
        triangle1[i] = Integer.parseInt(st.nextToken());
    }
    bw.write(isValidTraiangle(triangle1, triangle2) ? "YES\n" : "NO\n");
}
```
