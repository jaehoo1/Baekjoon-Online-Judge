# [27918 : 탁구 경기](https://www.acmicpc.net/problem/27918)
https://www.acmicpc.net/problem/27918

## 아이디어
Simulation(Implementation)

```python
N = int(input())
D, P = 0, 0
for _ in range(N):
    # 라운드 별 승자를 입력받고
    winner = input()
    # 점수 카운팅
    if winner == 'D':
        D += 1
    elif winner == 'P':
        P += 1
    # 2점 이상 차이나면 경기 종료
    if abs(D - P) >= 2:
        break
print(f"{D}:{P}")
```
