# [29666 : Блокировка телефона](https://www.acmicpc.net/problem/29666)
https://www.acmicpc.net/problem/29666

## 문제
![](https://upload.acmicpc.net/2854974b-c847-487a-b13b-b54021d1ac2f/)  
번호 키패드에서 3자리의 누른 번호가 주어질 때 같은 행/열에 있고 붙어있는 숫자들인지 판별하는 문제

## 아이디어
Implementation

눌렀다는 표시를 하는 `boolean[][]` 배열에 누른 수들을 `true`로 표시하고 붙어있는 지 확인하는 것을 구현하여 문제 해결
```java
boolean[][] visited = new boolean[4][3];
String s = new Scanner(System.in).next();
for (int i = 0; i < 3; i++) {
    // 행/열 처리하기 쉽게 -1 해줌
    int num = Integer.parseInt(String.valueOf(s.charAt(i))) - 1;
    // 0인 경우 예외 처리
    if (num < 0) {
        num = 10;
    }
    // 누른 좌표 저장
    visited[num / 3][num % 3] = true;
}

boolean unlocked = false;
// 행/열 숫자가 붙어있는지 확인
for (int i = 0; i < 3; i++) {
    int j;
    for (j = 0; j < 3 && visited[i][j]; j++);
    if (j == 3) {
        unlocked = true;
    }
    for (int k = 0; k <= 1; k++) {
        for (j = k; j < k + 3 && visited[j][i]; j++);
        if (j == k + 3) {
            unlocked = true;
        }
    }
}

System.out.println(unlocked ? "Unlocked" : "Locked");
```
