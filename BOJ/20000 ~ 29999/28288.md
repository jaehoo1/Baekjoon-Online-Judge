# [28288 : Special Event](https://www.acmicpc.net/problem/28288)
https://www.acmicpc.net/problem/28288

## 문제
사람이 가장 많이 출석할 수 있는 날들을 구하는 문제

## 아이디어
Brute Force, String, Implementation

입력에 대해 Y가 가장 많은 열에 Y가 몇 개 있는지(`max`) 찾고
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
int N = Integer.parseInt(br.readLine());
int max = 0;
char[][] attendances = new char[N][5];
for (int i = 0; i < N; i++) {
    attendances[i] = br.readLine().toCharArray();
}
for (int i = 0; i < 5; i++) {
    int cnt = 0;
    for (int j = 0; j < N; j++) {
        if (attendances[j][i] == 'Y') {
            cnt++;
        }
    }
    max = Math.max(max, cnt);
}
```

열 중 가장 많이 나온 Y랑 동일한 열은 출석이 가장 많이 가능한 날
```java
List<String> result = new ArrayList<>(5);

for (int i = 0; i < 5; i++) {
    int cnt = 0;
    for (int j = 0; j < N; j++) {
        if (attendances[j][i] == 'Y') {
            cnt++;
        }
    }
    if (cnt == max) {
        result.add(String.valueOf(i + 1));
    }
}
System.out.println(String.join(",", result));
```
