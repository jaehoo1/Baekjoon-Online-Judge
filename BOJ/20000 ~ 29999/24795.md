# [24795 : Saving Princess Peach](https://www.acmicpc.net/problem/24795)
https://www.acmicpc.net/problem/24795

## 문제
**문제**  
마리오가 사랑하는 피치 공주를 구하려 합니다! 하지만 그러기 위해 마리오는 수많은 장애물을 넘어야 합니다. 그래서 그는 쿠파의 성에 침투하기 위한 거창한 계획을 세웁니다.

하지만 먼저 연습이 필요합니다. 형제 루이지가 마리오를 훈련시키기 위한 연습 코스를 만들었습니다. 이 코스에서 마리오는 목숨을 빼앗을 수 있는 모든 장애물을 찾는 연습을 합니다. 그런데 마리오는 덜렁대서 일부 장애물을 놓치고, 일부는 두 번 이상 세고, 찾은 장애물의 순서도 뒤죽박죽입니다!

루이지가 마리오에게 어떤 장애물을 놓쳤는지 알려줄 수 있도록 프로그램을 작성하세요.

<br/>

**입력**  
첫 번째 줄에 $2$개의 값이 주어집니다. 첫 번째 값 $N$ ($0 < N \le 100$) 은 장애물의 총 개수입니다. 장애물 번호는 $0 \ldots N-1$입니다.

두 번째 값 $Y$ ($0 \le Y \le 200$)는 마리오가 연습 도중 찾았다고 말한 장애물의 수입니다. 이후 $Y$개의 줄에 각각 마리오가 찾았다고 말한 장애물 번호 $k$ ($0 \le k < N$) 가 하나씩 주어집니다.

<br/>

**출력**  
먼저 마리오가 놓친 장애물 번호를 오름차순으로 한 줄에 하나씩 출력합니다. 마지막 줄에는 `Mario got X of the dangerous obstacles.`를 출력합니다. 여기서 $X$는 마리오가 찾은 중복 없는 장애물의 수입니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/165138

## 아이디어
Implementation, Set

마리오가 말하는 찾은 장애물은 순서 뒤죽박죽, 중복이 가능할 수도 있음 → 중복제거 필요 → `Set<Integer>`

마리오가 말하는 찾은 장애물을 중복 제거하며 저장한다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

Set<Integer> set = new HashSet<>();
StringTokenizer st = new StringTokenizer(br.readLine());
int N = Integer.parseInt(st.nextToken());
int Y = Integer.parseInt(st.nextToken());
while (Y-- > 0) {
    set.add(Integer.parseInt(br.readLine()));
}
```

$N$까지 장애물 중 찾지 않는 것이 무엇인지 확인하여 출력한다.
```java
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

for (int i = 0; i < N; i++) {
    if (set.contains(i)) {
        continue;
    }
    bw.write(i + "\n");
}
bw.write("Mario got " + set.size() + " of the dangerous obstacles.");
```
