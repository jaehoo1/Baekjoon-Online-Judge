# [20282 : Game Show!](https://www.acmicpc.net/problem/20282)
https://www.acmicpc.net/problem/20282

## 문제
**문제**  
SBC(Society of Good Competitors)는 제휴 경쟁사를 위해 TV 쇼(지금은 온라인 스트리밍도 가능!)를 조직합니다. SBC는 sbecs라는 크레딧 시스템을 사용하여 대회에 참가하거나 각 시즌이 끝날 때 상금으로 교환할 수 있습니다. 그들은 새로운 유형의 게임을 시작했으며 상금 풀에서 큰 손실을 피하기 위해 몇 가지 시뮬레이션을 수행해야 합니다!

이 새로운 게임에 참여하려면 참가자는 100sbecs를 베팅해야 하며 이 금액은 자신의 게임 잔액으로 이전되고 일련의 상자가 배치됩니다. 게임은 라운드로 구성되며 최대 라운드 수는 상자 수와 같습니다. 각 라운드에서 플레이어는 다음 상자를 열지 아니면 게임을 종료할지 결정합니다. 그가 그만두면 그는 sbecs의 실행 균형을 되찾습니다. 상자를 열면 상자에 담긴 비밀 번호가 잔액에 추가되고 게임이 계속됩니다. 비밀 번호가 음수가 될 수 있기 때문에 매우 탐욕스러운 플레이어는 결국 패배할 수 있습니다! 플레이어가 게임을 종료하기로 결정하거나 마지막 상자가 열리면 게임이 종료됩니다.

SBC는 게임 테스트를 위해 당신을 고용했습니다. 상자의 내용물에서 플레이어가 얻을 수 있는 가장 높은 상품이 무엇인지 결정해야 합니다.

<br/>

**입력**  
입력의 첫 번째 줄에는 게임의 상자 수인 정수 C, 1 ≤ C ≤ 100이 포함됩니다. 그런 다음 다음 각 C 행은 C 상자의 내용을 순서대로 설명합니다. 그들 각각은 해당 상자의 내용에 해당하는 정수 V, −1000 ≤ V ≤ 1000을 포함합니다.

<br/>

**출력**  
출력은 '해당 상자 시퀀스에 대해 가능한 가장 높은 지불금'에 해당하는 정수를 포함하는 한 줄로 구성됩니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/120354

## 아이디어
Implementation

상자에서 나온 내용물을 적용해가며 최대값을 갱신해서 전체 턴별 최대값 구하기
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int C = Integer.parseInt(br.readLine());
int max = 100;
int cur = 100;
while (C-- > 0) {
    cur += Integer.parseInt(br.readLine());
    max = Math.max(max, cur);
}
System.out.println(max);
```
