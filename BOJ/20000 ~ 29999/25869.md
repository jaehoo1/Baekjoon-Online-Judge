## 문제
직사각형 벽의 너비와 높이와 창 테두리 간격 (벽의 둘레와 창 둘레 사이에 필요한 최소 거리)을 감안할 때 벽에 설치할 수있는 가장 큰 직사각형 창의 면적을 구하는 문제

<br/>

## 아이디어
Arithmetic

테두리 간격을 뺐을 때 음수라면(테두리 간격이 너비/높이보다 크다면)  
공간을 만들 수 없음
```c
int w, h, d, res;
scanf("%d%d%d", &w, &h, &d);
res=(w-2*d)*(h-2*d);
if(w-2*d<0||h-2*d<0) {
	puts("0");
	return 0;
}
printf("%d", res);
```
