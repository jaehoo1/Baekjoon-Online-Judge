## 아이디어
Arithmetic(Math), Implementation

도장 개수에 따라 적용할 수 있는 쿠폰들을 적용해보고 최소값을 찾는다.
```python
N = int(input())
P = int(input())
price = P
if N >= 5:
    price = min(price, P - 500)
if N >= 10:
    price = min(price, int(P * 0.9))
if N >= 15:
    price = min(price, P - 2000)
if N >= 20:
    price = min(price, int(P * 0.75))
if price < 0:
    price = 0
print(price)
```
