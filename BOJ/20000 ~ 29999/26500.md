# [26500 : Absolutely](https://www.acmicpc.net/problem/26500)
https://www.acmicpc.net/problem/26500

## 문제
두 점 사이의 거리(절대값)를 구해 소수점 한 자리 반올림 하는 문제

## 아이디어
Arithmetic(Math)

소수점 절대값, 반올림 계산 구현
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int n = Integer.parseInt(br.readLine());
while (n-- > 0) {
    StringTokenizer st = new StringTokenizer(br.readLine());
    bw.write(String.format("%.1f\n", Math.abs(Double.parseDouble(st.nextToken()) - Double.parseDouble(st.nextToken()))));
}
```
