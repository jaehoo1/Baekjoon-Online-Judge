# [29965 : Average scores](https://www.acmicpc.net/problem/29965)
https://www.acmicpc.net/problem/29965

## 문제
**문제**  
Mari와 Jüri는 컴퓨터 게임을 하며, 누가 평균 점수가 더 높은지 언쟁을 벌이고 있습니다. 두 사람의 결과를 바탕으로 누가 평균 점수가 더 높은지 결정하는 프로그램을 작성하세요.

<br/>

**입력**  
입력의 첫 번째 줄에는 정수 $N$이 주어집니다 $(1 \le N \le 500000)$.

다음 $N$개의 줄 각각에는 대문자 `M` 또는 `J`, 그리고 그 뒤에 공백으로 구분된 정수 $P$가 주어집니다 $(0 \le P \le 10000)$.

여기서 `M`은 Mari를, `J`는 Jüri를 나타내며, $P$는 해당 플레이어가 한 번의 플레이에서 얻은 점수입니다.

<br/>

**출력**  
출력은 오직 한 줄이며,

Mari의 평균 점수가 Jüri보다 높으면 대문자 `M`, Jüri의 평균 점수가 Mari보다 높으면 대문자 `J`, 두 사람의 평균 점수가 같으면 대문자 `V`를 출력해야 합니다.

만약 어떤 플레이어가 단 한 번도 게임을 하지 않았다면, 그 플레이어의 평균 점수는 $0$으로 간주합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/160996

## 아이디어
Arithmetic(Math), Implementation

단순히 평균 대소비교를 하는 문제인데, 0으로 나눌 때의 예외 처리만 해주면 된다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

double mari = 0, juri = 0;
int M = 0, J = 0;
int N = Integer.parseInt(br.readLine());
// 플레이어 별 합 구한 다음
while (N-- > 0) {
    StringTokenizer st = new StringTokenizer(br.readLine());
    String player = st.nextToken();
    int points = Integer.parseInt(st.nextToken());
    if (player.equals("M")) {
        mari += points;
        M++;
    } else {
        juri += points;
        J++;
    }
}
// 평균 구해서
mari = M == 0 ? 0 : mari / M;
juri = J == 0 ? 0 : juri / J;
// 대소비교
if (mari > juri) {
    System.out.println("M");
} else if (mari < juri) {
    System.out.println("J");
} else {
    System.out.println("V");
}
```
