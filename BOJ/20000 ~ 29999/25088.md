# [25088 : Punched Cards](https://www.acmicpc.net/problem/25088)
https://www.acmicpc.net/problem/25088

## 문제
별찍기 확장판

## 아이디어
Implementation

단계별로 cell들을 채우면 된다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int T = Integer.parseInt(br.readLine());
for (int t = 1; t <= T; t++) {
    StringTokenizer st = new StringTokenizer(br.readLine());
    int R = Integer.parseInt(st.nextToken());
    int C = Integer.parseInt(st.nextToken());

    // 초기화: 모든 셀들을 '.' 으로 채움
    char[][] cells = new char[2 * R + 1][2 * C + 1];
    for (int i = 0; i < 2 * R + 1; i++) {
        Arrays.fill(cells[i], '.');
    }

    // '+' 채우기
    for (int i = 0; i < 2 * R + 1; i += 2) {
        for (int j = 0; j < 2 * C + 1; j += 2) {
            cells[i][j] = '+';
        }
    }

    // '-' 채우기
    for (int i = 0; i < 2 * R + 1; i += 2) {
        for (int j = 1; j < 2 * C + 1; j += 2) {
            cells[i][j] = '-';
        }
    }

    // '|' 채우기
    for (int i = 1; i < 2 * R + 1; i += 2) {
        for (int j = 0; j < 2 * C + 1; j += 2) {
            cells[i][j] = '|';
        }
    }

    // 왼쪽 위 네 칸은 전부 '.'
    cells[0][0] = cells[0][1] = cells[1][0] = cells[1][1];

    // 출력
    bw.write("Case #" + t + ":\n");
    for (int i = 0; i < 2 * R + 1; i++) {
        for (int j = 0; j < 2 * C + 1; j++) {
            bw.write(cells[i][j]);
        }
        bw.write("\n");
    }
}
```
