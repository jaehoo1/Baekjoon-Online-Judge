# [25278 : Terraforming](https://www.acmicpc.net/problem/25278)
https://www.acmicpc.net/problem/25278

## 문제
세계 정부의 대변인 레비 유켄(2315년)에 따르면, 화성을 살 수 있는 행성으로 바꾸려는 인류의 노력은 "엄청난 도전으로 완수를 위해서는 인류의 단합된 노력이 필요"합니다.

이 도전을 시작할 때, 화성에는 바다도 산소도 없었으며 평균 기온은 $-30$도였습니다.

물

물의 순환, 습도, 온도, 일교차의 안정화를 위해서 화성은 $9\%$ 의 바다 표면적을 필요로 합니다.

기온

아주 낮은 기온에서, 한 해의 대부분 바다는 빙하 상태입니다. 이미 '더운' 여름 21세기 화성의 기온은 $20$도까지 오르지만, 평균 기온은 어는 점보다 한참 아래입니다. 바다가 액체 상태로 존재하기 위해 최소한 적도에서는 기온이 양수가 돼야 하며, 이는 $+8$ 도의 평균 기온을 필요로 함을 말합니다.

산소

화성을 살 수 있는 행성으로 바꾸기 위해 가장 중요한 것은 산소입니다 - 숨을 쉴 수 있는 공기가 없다면, 사람은 화성에서 살 수 없습니다. 해수면에서 지구의 산소 농도는 공기의 $21\%$ 입니다. 하지만 $21$세기 거의 100만 명의 인구가 살고 있는 볼리비아의 도시 엘알토($4150$ m)나 라파즈($3640$ m)를 고려할 대, $14\%$ 면 사람이 살기에 충분합니다.

<br/>

**입력**  
첫 줄에 화성에 가할 변화의 수 $n\in\{1,\ldots, 315\}$ 가 주어집니다. 다음 $n$개의 줄 각각에 변화를 가할 요소(oxygen, ocean, temperature)와 하나의 정수가 주어집니다.

산소와 물에 가하는 변화는 퍼센트 포인트로, 기온에 가하는 변화는 섭도로 나타냅니다.

<br/>

**출력**  
환경을 변화해 바다 표면적이 최소 $9\%$, 산소가 최소 $14\%$, 온도가 최소 $+8$도가 보장됐다면, 'liveable'을 출력합니다. 그 외의 경우, 'not liveable'을 출력합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/91945

## 아이디어
String, Implementation

입력되는 문자열 (파싱) 분기 처리를 통해 물/산소/온도 분기처리 연산 후 최종 결과가 liveable 조건을 만족하는지 확인
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int water = 0;
int temperature = -30;
int oxygen = 0;
int n = Integer.parseInt(br.readLine());
while (n-- > 0) {
    StringTokenizer st = new StringTokenizer(br.readLine());
    String parameter = st.nextToken();
    int integer = Integer.parseInt(st.nextToken());
    if ("ocean".equals(parameter)) {
        water += integer;
    } else if ("temperature".equals(parameter)) {
        temperature += integer;
    } else if ("oxygen".equals(parameter)) {
        oxygen += integer;
    }
}
System.out.println((water >= 9 && oxygen >= 14 && temperature >= 8) ? "liveable" : "not liveable");
```
