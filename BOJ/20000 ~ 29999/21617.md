# [21617 : Crazy Fencing](https://www.acmicpc.net/problem/21617)
https://www.acmicpc.net/problem/21617

## 문제
사다리꼴 여러 개의 변 길이들이 주어질 때 면적 합을 구하는 문제

## 아이디어
Geometry(Math)

윗변 a, 밑변 b, 높이 h인 사다리꼴의 면적을 구하는 공식은 다음과 같다.
```java
static double areaOfTrapezoid(int a, int b, int h) {
    return (double) (a + b) * h / 2;
}
```

변 길이들을 입력받는다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
int N = Integer.parseInt(br.readLine());
int[] heights = new int[N + 1];
int[] widths = new int[N];
StringTokenizer st = new StringTokenizer(br.readLine());
for (int i = 0; i < N + 1; i++) {
    heights[i] = Integer.parseInt(st.nextToken());
}
st = new StringTokenizer(br.readLine());
for (int i = 0; i < N; i++) {
    widths[i] = Integer.parseInt(st.nextToken());
}
```

각 사다리꼴 별 면적을 구하고 합을 구한다.
```java
double totalArea = 0;
for (int i = 0; i < N; i++) {
    totalArea += areaOfTrapezoid(heights[i], heights[i + 1], widths[i]);
}
System.out.printf("%.7f", totalArea);
```
