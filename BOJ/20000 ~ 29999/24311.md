## 문제
프로그래밍 대회가 있기 전에 다양한 도시에서 오는 참가 팀의 팀장을 상대로 기술 세미나가 열립니다. 기술 세미나의 시작 시간은 대회 프로그램 설명 시 발표되며 각 팀장은 호텔에 짐을 풀고, 참가자를 등록하고 난 후 기술 세미나가 열리기 10분 전까지 홀에 도착할 수 있도록 팀의 여행 일정을 짜려 합니다.

기술 세미나의 시작 시간 (시와 분으로 주어집니다.), 참가자를 등록하는 데 걸리는 총 시간 (분으로 주어집니다.), 각 팀원이 호텔에 짐을 푸는 데 걸리는 시간 (분으로 주어집니다.), 참가자가 있는 곳으로부터 대회 장소까지 이동하는 데 걸리는 시간 (시와 분으로 주어집니다.), 팀장을 제외한 팀원의 수를 입력 받아, 팀장이 기술 세미나에 늦지 않게 위해 최소 몇 시 몇 분에 출발해야 하는지 출력하는 여행 계획 프로그램을 작성합시다. 각 그룹에는 한 명의 팀장이 있으며, 여행은 무조건 기술 세미나가 있는 날 출발해야 합니다.

<br/>

**입력**  
첫 줄에 하나의 공백으로 나뉘어진 두 정수 t1과 t2가 주어집니다 - 이는 기술 세미나가 시작하는 시각의 시간, 분입니다.

둘째 줄에 정수 t3이 주어집니다 - 참가자 등록에 걸리는 시간을 분 단위로 나타낸 것입니다.

셋째 줄에 하나의 공백으로 나뉘어진 두 정수 t4, t5가 주어집니다 - 이는 이동에 걸리는 시간 및 분을 나타낸 것입니다.

넷째 줄에 자연수 br이 주어집니다 - 이는 팀장을 제외한 팀원의 수입니다.

다섯째 줄에 자연수 t6가 주어집니다 - 이는 한 명의 학생이 호텔에 짐을 푸는 데 걸리는 시간을 분으로 나타낸 것입니다.

<br/>

**출력**  
오로지 한 줄에 두 개의 정수를 출력합니다 - 최소 몇 시 몇 분에 출발해야 팀장이 기술 세미나에 늦지 않는지 나타내는 두 정수입니다. 숫자는 하나의 공백으로 구분되며 불필요한 0 역시 출력해야 합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/86302

## 아이디어
Arithmetic(Math), Implementation

python `datetime` 모듈의 `datetime`, `time`, `timedelta`를 사용해 시간 연산을 쉽게할 수 있다.
> 문제 조건이 당일이니 날짜 변경도 무시할 수 있다.
```python
from datetime import datetime, time, timedelta

t1, t2 = map(int, input().split())
t3 = int(input())
t4, t5 = map(int, input().split())
br = int(input())
t6 = int(input())

start_time = datetime.combine(datetime.today(), time(t1, t2))   # 아무날짜 + 입력시간
start_time -= timedelta(minutes=10)                             # 기술 세미나가 열리기 10분 전
start_time -= timedelta(minutes=t3)                             # 참가자 등록에 걸리는 시간
start_time -= timedelta(hours=t4, minutes=t5)                   # 이동에 걸리는 시간 및 분
start_time -= timedelta(minutes=(br + 1) * t6)                  # 팀장을 포함해 호텔에 짐을 푸는 데 걸리는 시간
print(start_time.strftime("%H %M"))                             # date.strftime 을 통해 0 padding을 쉽게 할 수 있다.
```
