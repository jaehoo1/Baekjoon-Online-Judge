# [29684 : Which Team Should Receive the Sponsor Prize?](https://www.acmicpc.net/problem/29684)
https://www.acmicpc.net/problem/29684

## 문제
어떤 팀이 2023과 가장 차이가 적은지 찾는 문제

## 아이디어
Implementation

2023과의 차이를 구하는 로직을 메서드 추출 해둔다.
```java
static int diff(int elapsedTime) {
    return Math.abs(2023 - elapsedTime);
}
```

입력되는 팀 마다 2023과의 차이를 구해 제일 2023에 가까운 팀을 찾는다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
while (true) {
    int n = Integer.parseInt(br.readLine());
    if (n == 0) {
        break;
    }
    int minDiff = Integer.MAX_VALUE;
    int winner = 0;
    StringTokenizer st = new StringTokenizer(br.readLine());
    for (int i = 0; i < n; i++) {
        int elapsedTime = Integer.parseInt(st.nextToken());
        int diff = diff(elapsedTime);
        if (diff < minDiff) {
            minDiff = diff;
            winner = i + 1;
        }
    }
    bw.write(winner + "\n");
}
```
