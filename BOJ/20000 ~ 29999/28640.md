## 문제
**문제**  
당신과 친구들은 \<\<Among Us\>\>를 플레이하는 데 동의했습니다. 언제나 늦는 Arseny를 제외한 모든 사람들이 정해진 시간에 게임에 입장했습니다. 대기실에 있는 동안에는 최근 게임 업데이트에서 실험적으로 추가된 가상 도미노를 플레이하기로 결정했습니다.

모드는 아직 실험적이므로 게임은 원시적입니다. 테이블 위에 두 개의 도미노가 있습니다. 각 도미노는 두 부분으로 나누어진 직사각형입니다. 각각의 도미노는 둘로 나뉘어 있고 각자 0~6개의 점이 있습니다. 플레이어는 이 두 도미노를 결합할 수 있는지 여부를 결정해야 합니다. 음수가 아닌 정수가 존재하는 경우에만 도미노를 결합할 수 있습니다. 적어도 각 도미노가 나뉘어진 두 부분 중 한 쪽에 동일하게 $k$개의 점이 그려져 있는 경우.

이 게임은 너무 원시적이어서 자동으로 결합하는 프로그램을 작성하기로 결정했습니다.

<br/>

**입력**  
입력에는 두 줄이 포함되어 있으며 각 줄에는 도미노에 대한 설명이 포함되어 있습니다. 도미노의 설명은 여러 개의 점(기호 \<\< . \>\>, ASCII 코드 $46$) 으로 구성됩니다. 및 하나의 수직 막대(문자 \<\< | \>\>, ASCII 코드 124)로 구성됩니다. 수직 막대를 기준으로 앞쪽에 있는 점과 뒤쪽에 있는 점이 둘로 나뉘어집니다.

각 도미노의 두 부분에는 각각 $0$~$6$개의 점이 포함되어 있음이 보장됩니다. 

<br/>

**출력**  
도미노를 합칠 수 있으면 `Yes`를 인쇄하고, 그렇지 않으면 `No`를 인쇄하세요.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/140045

## 아이디어
String, Implementation

'|' 기준으로 왼쪽과 오른쪽의 점 개수가 같은지를 판별한다.
```python
domino1 = input()
domino2 = input()

left1 = domino1.find('|')
right1 = len(domino1) - left1 - 1
left2 = domino2.find('|')
right2 = len(domino2) - left2 - 1

print("Yes" if left1 == left2 or left1== right2 or right1 == left2 or right1 == right2 else "No")
```
