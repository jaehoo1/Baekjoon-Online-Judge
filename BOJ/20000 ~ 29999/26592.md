# [26592 : Triangle Height](https://www.acmicpc.net/problem/26592)
https://www.acmicpc.net/problem/26592

## 문제
삼각형의 면적과 밑변의 길이가 주어질 때, 높이를 구하는 문제

## 아이디어
Geometry, Arithmetic(Math)

삼각형 넓이 공식 (넓이 = 밑변 × 높이 / 2) 으로 높이를 구할 수 있다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int n = Integer.parseInt(br.readLine());
while (n-- > 0) {
    StringTokenizer st = new StringTokenizer(br.readLine());
    double area = Double.parseDouble(st.nextToken());
    double base = Double.parseDouble(st.nextToken());
    bw.write("The height of the triangle is " + String.format("%.2f", area * 2 / base) + " units\n");
}
```
