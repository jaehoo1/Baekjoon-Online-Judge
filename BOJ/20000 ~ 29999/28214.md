# [28214 : 크림빵](https://www.acmicpc.net/problem/28214)
https://www.acmicpc.net/problem/28214

## 아이디어
Implementation

묶음별 빵 중 크림이 없는 빵의 개수를 세서 $P$ 미만인지 비교
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int sellable = 0;
StringTokenizer st = new StringTokenizer(br.readLine());
int N = Integer.parseInt(st.nextToken());
int K = Integer.parseInt(st.nextToken());
int P = Integer.parseInt(st.nextToken());
st = new StringTokenizer(br.readLine());
for (int i = 0; i < N; i++) {
    int nonCream = 0;
    for (int j = 0; j < K; j++) {           // 묶음별
        if ("0".equals(st.nextToken())) {   // 크림 없는 빵 개수 세기
            nonCream++;
        }
    }
    if (nonCream < P) {                     // 묶음별 크림 없는 빵 개수가 P 미만이여야 한다
        sellable++;
    }
}
System.out.println(sellable);
```
