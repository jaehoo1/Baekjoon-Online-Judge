# [24544 : 카카오뷰 큐레이팅 효용성 분석](https://www.acmicpc.net/problem/24544)

## 아이디어
Arithmetic(Math), Implementation

첫째 줄에는 전체 콘텐츠의 흥미도의 합을 출력
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int N = Integer.parseInt(br.readLine());
int[] interestScores = new int[N];
int sum = 0;
StringTokenizer st = new StringTokenizer(br.readLine());
for (int i = 0; i < N; i++) {
    interestScores[i] = Integer.parseInt(st.nextToken());
    sum += interestScores[i];
}
System.out.println(sum);
```

둘째 줄에는 My뷰에 등록되어있지 않은 콘텐츠들의 흥미도의 합을 출력  
→ 전체 합에서 My뷰에 등록된 것을 뺌
```java
st = new StringTokenizer(br.readLine());
for (int i = 0; i < N; i++) {
    if (st.nextToken().equals("1")) {
        sum -= interestScores[i];
    }
}
System.out.println(sum);
```
