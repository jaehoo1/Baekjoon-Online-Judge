# [28225 : Flower Festival](https://www.acmicpc.net/problem/28225)
https://www.acmicpc.net/problem/28225

## 문제
자동차 경주를 한다. 전체 도로의 길이와 각 자동차들의 현재 위치, 속도가 주어질 때 어떤 차가 우승하는지 구하는 문제

## 아이디어
Arithmetic(Math), Implementation

자동차가 우승하는데 걸리는 시간은 남은 거리 / 속력으로 구할 수 있다.
```java
/**
 * 남은 거리
 * 
 * @param lenOfStreet 도로 전체 길이
 * @param location 자동차 위치
 * @return 자동차에게 남은 길이
 */
static int remainedDistance(int lenOfStreet, int location) {
    return lenOfStreet - location;
}

/**
 * 남은 시간
 * 
 * @param remainedDistance 자동차가 가야할 거리
 * @param speed 자동차 속력
 * @return 완주까지 걸리는 시간
 */
static double remainedTime(int remainedDistance, int speed) {
    return (double) remainedDistance / speed;
}
```

시간이 가장 짧게 걸리는 차를 찾으면 된다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
StringTokenizer st = new StringTokenizer(br.readLine());
int n = Integer.parseInt(st.nextToken());       // 자동차 수
int f = Integer.parseInt(st.nextToken());       // 도로 길이
double min = Double.MAX_VALUE;
int num = 0;
for (int i = 1; i <= n; i++) {
    st = new StringTokenizer(br.readLine());
    int x = Integer.parseInt(st.nextToken());   // 자동차 위치
    int v = Integer.parseInt(st.nextToken());   // 자동차 속도
    double time = remainedTime(remainedDistance(f, x), v);  // 완주까지 걸리는 시간
    // 완주 시간 가장 짧은 자동차 찾기
    if (time < min) {
        min = time;
        num = i;
    }
}
System.out.println(num);
```
