## 문제
**문제**  
메트로놈은 음악가들이 박자를 맞추는 데 사용하는 기계 장치입니다. 이는 스프링, 역진자, 탈진 기어를 기반으로 한 매우 영리한 장치입니다. 마일로는 글로켄슈필 연주를 배우고 있으며, 박자를 맞추는 데 도움을 받고자 메트로놈을 구입했습니다. 마일로는 키를 한 번 완전히 돌릴 때마다(한 바퀴 회전) 메트로놈이 4번의 틱을 낸다는 것을 알아챘습니다. 마일로는 자신이 연주하려는 각 노래가 끝날 때 메트로놈이 멈추기를 원합니다.

주어진 노래에 대해, 그는 키를 몇 바퀴 감아야 할까요?

<br/>

**입력**  
입력의 한 줄에는 하나의 정수 $n$ ($1 \le n \le 10^5$)이 포함되어 있으며, 이는 노래의 길이를 틱으로 나타낸 것입니다.

<br/>

**출력**  
마일로가 노래가 정확히 끝날 때 메트로놈이 멈추도록 키를 돌려야 하는 회전 수를 나타내는 하나의 실수를 출력하세요.

이 숫자는 소수점 둘째 자리까지 정확해야 합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/146503

## 아이디어
Arithmetic(Math)

나눗셈연산 & 소수점 정확도 (`.3f`하면 틀림)
```python
print(f"{int(input())/4:.2f}")
```
