# [26550 : Ornaments](https://www.acmicpc.net/problem/26550)
https://www.acmicpc.net/problem/26550

## 문제
피라미드 층이 주어질 때 총 몇 개의 ornaments 로 이뤄지는지 구하는 문제

## 아이디어
Arithmetic(Math)

층이 늘 때 마다 (n층이라 할 때) 1 ~ n ornament 만큼 늘어난다.  
즉, 1은 n개, 2는 n-1개, ..., n은 1개 로 이뤄진다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int n = Integer.parseInt(br.readLine());
while (n-- > 0) {
    long cnt = 0;
    int num = Integer.parseInt(br.readLine());
    for (int i = 1; i <= num; i++) {
        cnt += (long) i * (num - i + 1);
    }
    bw.write(cnt + "\n");
}
```
