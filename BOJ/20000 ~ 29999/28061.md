# [28061 : 레몬 따기](https://www.acmicpc.net/problem/28061)
https://www.acmicpc.net/problem/28061

## 아이디어
Arithmetic(Math)

$i$번째 나무에서 레몬을 따면 집에 도착할 때의 레몬 개수는 $- N + i$ 를 더해줘야 한다. 이 최대값을 찾아주면 된다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int N = Integer.parseInt(br.readLine());
StringTokenizer st = new StringTokenizer(br.readLine());
int max = 0;
for (int i = 0; i < N; i++) {
    int lemon = Integer.parseInt(st.nextToken()) - N + i;
    max = Math.max(max, lemon);
}
System.out.println(max);
```
