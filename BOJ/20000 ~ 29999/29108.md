https://www.acmicpc.net/problem/29108

## 문제
입력받은 문자열이 "io" 뒤에 숫자가 오는 문자열인지 판별하는 문제

## 아이디어
String, Implementation

정규식 매칭을 활용하면 된다.

```java
System.out.println(new Scanner(System.in).next().matches("^io[0-9]+$") ? "Correct" : "Incorrect");
```

`^io[0-9]+$`
- `^io` : 'io'로 시작하는지
- `[0-9]+$` : 1자리 이상 숫자가 반복됨
