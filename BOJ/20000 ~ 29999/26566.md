# [26566 : Pizza](https://www.acmicpc.net/problem/26566)
https://www.acmicpc.net/problem/26566

## 문제
조각 피자의 면적과 가격, 한 판 피자의 반지름과 가격이 주어질 때 어떤 피자가 더 이득인지 구하는 문제

## 아이디어
Geometry, Arithmetic(Math)

면적당 가격을 구해 어떤 것이 더 싼지 비교한다
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int n = Integer.parseInt(br.readLine());
while (n-- > 0) {
    StringTokenizer st = new StringTokenizer(br.readLine());
    double A1 = Integer.parseInt(st.nextToken());
    double P1 = Integer.parseInt(st.nextToken());
    st = new StringTokenizer(br.readLine());
    double R1 = Integer.parseInt(st.nextToken());
    double P2 = Integer.parseInt(st.nextToken());
    // 조각피자 면적당 가격 vs 원피자 면적당 가격
    bw.write((A1 / P1) > (R1 * R1 * Math.PI / P2) ? "Slice of pizza\n" : "Whole pizza\n");
}
```
