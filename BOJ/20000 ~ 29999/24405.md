# [24405 : Eye of Sauron](https://www.acmicpc.net/problem/24405)
https://www.acmicpc.net/problem/24405

## 문제
`|`, `()`(eye) 로만 구성된 문자열이 주어질 때 `()`의 좌우 `|` 의 개수가 같은지 구하는 문제

## 아이디어
String, Implementation

`()` 옆의 `|` 개수가 같은지를 그대로 구현해도 되지만
```java
String s = new Scanner(System.in).nextLine();
int index = s.indexOf("()");
System.out.println(index == s.length() - index - 2 ? "correct" : "fix");
```

문제 조건 때문에 `()`는 가운데에 와야하니, `()`의 위치 만으로도 정답 여부를 확인할 수 있다.
```java
String s = new Scanner(System.in).nextLine();
System.out.println(s.indexOf("()") + 1 == s.length() / 2 && s.length() % 2 == 0 ? "correct" : "fix");
```
