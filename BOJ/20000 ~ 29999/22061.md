# [22061 : Покупка велосипеда](https://www.acmicpc.net/problem/22061)
https://www.acmicpc.net/problem/22061

## 문제
오늘 페트야는 방을 청소하다 방 아래에서 많은 잡동사니를 발견했습니다 - 1루블과 2루블 동전을 포함해서 말이죠. 동전을 유심히 센 결과, 페트야는 1루블 동전 a개와 2루블 동전 b개가 있음을 확인했습니다.

페트야는 아주 예전부터 자전거를 사고 싶어했기에, 당장 가게로 달려가 원했던 자전거의 가격을 확인했습니다. 안타깝게도 자전거 가게에는 거스름돈을 줄 작은 돈 단위가 부족하고, 페트야는 거스름돈을 받기 전까지는 가게를 떠나지 않으려 합니다. 따라서 페트야는 질문합니다: 자전거를 거스름돈을 내지 않고 살 수 있나요?

a개의 1루블 동전과 b개의 2루블 동전이 있다고 할 때, 자전거를 거스름돈 없이 구매할 수 있는지 말해 페트야를 도와주세요.

<br/>

**입력**  
첫 줄에 테스트 케이스의 수를 나타내는 하나의 정수 t(1 <= t <= 100000)가 주어집니다. 다음 t개의 줄 각각에 하나의 테스트 케이스가 주어집니다. 각 테스트 케이스는 세 정수 a, b, c로 이루어져 있습니다. (0 <= a, b, c <= 10 ^ 8) - 이는 각각 1루블 동전과 2루블 동전의 수, 자전거의 가격입니다.

<br/>

**출력**  
각 테스트 케이스에 대해, 한 줄을 출력합니다: 거스름돈 없이 자전거를 살 수 있으면 "YES"를, 그렇지 않은 경우 "NO"를 출력합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/91072

## 아이디어
Arithmetic(Math)

2루블 짜리 동전을 낼 수 있을 만큼 최대한 내고 나머지를 1원으로 낼 수 있는지를 확인해보면 된다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int t = Integer.parseInt(br.readLine());
while (t-- > 0) {
    StringTokenizer st = new StringTokenizer(br.readLine());
    long a = Long.parseLong(st.nextToken());
    long b = Long.parseLong(st.nextToken());
    long c = Long.parseLong(st.nextToken());
    bw.write((c - 2 * Math.min(c / 2, b) <= a) ? "YES\n" : "NO\n");
}
```
