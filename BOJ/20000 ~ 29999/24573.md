# [24573 : Good Fours and Good Fives](https://www.acmicpc.net/problem/24573)
https://www.acmicpc.net/problem/24573

## 문제
N을 4와 5의 합으로 나타낼 수 있는 경우의 수를 구하는 문제

## 아이디어
Brute Force, Math

4로(5로) 나눈 나머지가 5로(4로) 나누어 떨어지면 4와 5의 합으로 만들 수 있다.
```java
int N = new Scanner(System.in).nextInt();

int cnt = 0;
for (int i = 0; 4 * i <= N; i++) {
    if ((N - 4 * i) % 5 == 0) {
        cnt++;
    }
}
System.out.println(cnt);
```
