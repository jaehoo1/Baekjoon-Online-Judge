## 문제
피자 조각 수들이 주어지면 몇 판이 필요한지 구하는 문제

## 아이디어
Arithmetic(Math), Implementation

입력되는 피자 조각들을 전부 세어준다.
```python
from collections import Counter

n = int(input())
slices = Counter()
for _ in range(n):
    s, l = input().split()
    l = int(l)
    slices[s] += l
```

나누어 떨어지지 않는 피자도 세어줘야 하니 나머지를 더해서 몫을 구한다.
```python
slices['S'] += 5
slices['M'] += 7
slices['L'] += 11
print(slices['S'] // 6 + slices['M'] // 8 + slices['L'] // 12)
```
