# [32587 : Dragged-out Duel](https://www.acmicpc.net/problem/32587)
https://www.acmicpc.net/problem/32587

## 문제
가위바위보 2명이서 했을 때 둘 중 누가 더 많이 이겼는지 구하는 문제

## 아이디어
String, Case Work(Implementation)

가위바위보 승자 조건분기를 구현하면 된다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
int n = Integer.parseInt(br.readLine());
char[] me = br.readLine().toCharArray();
char[] guile = br.readLine().toCharArray();
int win = 0, lose = 0;
for (int i = 0; i < n; i++) {
    if (me[i] == guile[i]) {
        continue;
    }
    if (me[i] == 'R') {
        if (guile[i] == 'P') {
            lose++;
        } else if (guile[i] == 'S') {
            win++;
        }
    } else if (me[i] == 'P') {
        if (guile[i] == 'R') {
            win++;
        } else if (guile[i] == 'S') {
            lose++;
        }
    } else if (me[i] == 'S') {
        if (guile[i] == 'R') {
            lose++;
        } else if (guile[i] == 'P') {
            win++;
        }
    }
}
System.out.println(win > lose ? "victory" : "defeat");
```
