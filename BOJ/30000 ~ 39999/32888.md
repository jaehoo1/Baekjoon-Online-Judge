## 문제
직각삼각형 (작은) 두 변의 길이가 주어지면 나머지 한 변의 길이를 구하는 문제

## 아이디어
피타고라스 정리, Geometry, Arithmetic, Math

직각삼각형 $a^2 + b^2 = c^2$
```java
Scanner sc = new Scanner(System.in);
double a = sc.nextDouble();
double b = sc.nextDouble();
System.out.printf("%.9f", Math.sqrt(a * a + b * b));
```
