## 문제
**문제**  
![](https://u.acmicpc.net/c3a2639a-5d4b-4134-8aeb-82f13cacd422/tile_nwerc.jpg)  
정사각형 델프트 타일의 전형적인 원형 디자인.

델프트는 파란색과 흰색 도자기(델프트 블라우라고도 함)로 세계적으로 유명하며, 가장 초기의 작품은 16세기로 거슬러 올라갑니다.

델프트에서 산책하던 크리스티안은 기념품 가게에서 접시, 타일, 꽃병과 같은 아름답게 칠해진 도자기 물건을 많이 봅니다. 그런 것에서 영감을 받은 크리스티안은 직접 사각형 타일을 손으로 그리기로 결심합니다. 그는 이미 타일 중앙에 배치하고 싶은 원형 디자인을 선택했습니다. 그는 디자인 주변에 최소 $1cm$의 여백을 남기고 싶어합니다. 이제 크리스티안은 적절한 크기의 사각형 타일만 구입하면 됩니다.

원형 디자인의 면적을 고려할 때, 정사각형 타일의 최소 면적은 얼마입니까?

<br/>

**입력**  
입력은 텍스트 ${cm}^2$에서 원형 디자인의 면적을 정수 $a$ ($1 \leq a\leq 10^{15}$)로 나눈 한 줄로 구성됩니다.

<br/>

**출력**  
크리스티아안이 디자인 주위에 최소 $1cm$의 여백을 남긴 면적 $a$의 디자인에 필요한 정사각형 타일의 최소 면적을 ${cm}^2$로 출력합니다.

정답의 절대 또는 상대 오차는 $10^{-9}$입니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/161317

## 아이디어
Geometry(Arithmetic, Math)

원 면적(`a`)으로 반지름(`r`)을 구해 정사각형 면적의 넓이($(2(r + 1))^2$)를 구할 수 있다.
```python
import math

a = int(input())
r = math.sqrt(a / math.pi)
print(f"{(2 * (r + 1)) ** 2:.10f}")
```
