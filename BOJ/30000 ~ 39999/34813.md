## 아이디어

입력 문자열의 첫 문자에 따른 분기처리
```python
char = input()[0]
if char == 'F':
    subject = "Foundation"
elif char == 'C':
    subject = "Claves"
elif char == 'V':
    subject = "Veritas"
elif char == 'E':
    subject = "Exploration"
print(subject)
```
