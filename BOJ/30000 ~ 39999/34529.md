# [34529 : Acquiring SW-IT Corn](https://www.acmicpc.net/problem/34529)
https://www.acmicpc.net/problem/34529

## 아이디어
Arithmetic(Math)

각 스위트콘 별로 금액을 구해 더하면 된다.
```java
Scanner sc = new Scanner(System.in);
int X = sc.nextInt();
int Y = sc.nextInt();
int Z = sc.nextInt();
int U = sc.nextInt();
int V = sc.nextInt();
int W = sc.nextInt();
System.out.println(U / 100 * X + V / 50 * Y + W / 20 * Z);
```
