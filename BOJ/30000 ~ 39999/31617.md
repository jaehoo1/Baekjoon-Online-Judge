# [31617 : 差 (Difference)](https://www.acmicpc.net/problem/31617)
https://www.acmicpc.net/problem/31617

## 문제
정수 K 와 길이가 N 인 수열 A=(A1,A2,…,AN) 및 길이가 M 인 수열 B=(B1,B2,…,BM) 가 주어져 있다.

다음의 조건을 모두 만족하는 2 개의 정수로 이루어진 쌍 (p,q) 의 개수를 구해라.

- 1 ≦ p ≦ N．
- 1 ≦ q ≦ M．
- Ap + K = Bq．

<br/>

**입력**  
입력은 아래와 같은 형식으로 주어진다.
```text
K
N
A_1   A_2   …   A_N
M
B_1   B_2   …   B_N
```

<br/>

**출력**  
조건을 모두 만족하는 2 개의 정수로 이루어진 쌍 (p,q) 의 개수를 출력해라

<br/>

**제한**  
- 1 ≦ K ≦ 100．
- 1 ≦ N ≦ 100．
- 1 ≦ M ≦ 100．
- 1 ≦ Ai ≦ 100 (1 ≦ i ≦ N)．
- 1 ≦ Bj ≦ 100 (1 ≦ j ≦ M)．
- 입력으로 주어지는 값은 모두 정수.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/152980

## 아이디어
Brute Force, Arithmetic(Math), Implementation

N, M 제한이 작기 때문에 이중 `for` 루프로 문제를 풀 수 있다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int K = Integer.parseInt(br.readLine());
int N = Integer.parseInt(br.readLine());
int[] A = new int[N];
StringTokenizer st = new StringTokenizer(br.readLine());
for (int i = 0; i < N; i++) {
    A[i] = Integer.parseInt(st.nextToken());
}
int M = Integer.parseInt(br.readLine());
int[] B = new int[M];
st = new StringTokenizer(br.readLine());
for (int i = 0; i < M; i++) {
    B[i] = Integer.parseInt(st.nextToken());
}
int cnt = 0;
for (int i = 0; i < N; i++) {
    for (int j = 0; j < M; j++) {
        if (A[i] + K == B[j]) {
            cnt++;
        }
    }
}
System.out.println(cnt);
```
