# [31694 : Kto wygrał?](https://www.acmicpc.net/problem/31694)
https://www.acmicpc.net/problem/31694

## 문제
Algosia Bajtek 둘이 알고리즘 점수 경쟁을 한다. 각자 18개의 문제를 푸는데 점수 총합이 높은 사람이 이기게 된다. 점수 총합이 동일할 경우, 10점 문제를 많이 푼 사람이 이기고, 그 개수가 같으면 9점 문제를 많이 푼 사람이 이기고, ...

누가 이기는지 구하는 문제

## 아이디어
Implementation

점수 입력 시 합과 점수 개수도 같이 카운팅 해준다.
```java
Scanner sc = new Scanner(System.in);
int[] algosia = new int[11];
int[] bajtek = new int[11];
int asum = 0, bsum = 0;
for (int i = 0; i < 18; i++) {
    int score = sc.nextInt();
    algosia[score]++;
    asum += score;
}
for (int i = 0; i < 18; i++) {
    int score = sc.nextInt();
    bajtek[score]++;
    bsum += score;
}
```

합이 높은 사람이 이기고, 합이 같을 경우는 높은 점수 문제를 누가 더 많이 풀었는지 찾는다.
```java
if (asum > bsum) {
    System.out.println("Algosia");
    System.exit(0);
} else if (asum < bsum) {
    System.out.println("Bajtek");
    System.exit(0);
} else {
    for (int i = 10; i > 0; i--) {
        if (algosia[i] > bajtek[i]) {
            System.out.println("Algosia");
            System.exit(0);
        } else if (algosia[i] < bajtek[i]) {
            System.out.println("Bajtek");
            System.exit(0);
        }
    }
}
System.out.println("remis");
```
