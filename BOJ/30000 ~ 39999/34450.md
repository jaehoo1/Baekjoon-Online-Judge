# [34450 : Partial Transmission](https://www.acmicpc.net/problem/34450)
https://www.acmicpc.net/problem/34450

## 문제
$p$ ~ $p + n$ 까지 수들 중 빠진 수를 찾는 문제

## 아이디어
Arithmetic(Math), Implementation

수를 입력받으면서 `boolean[]`에 존재 여부를 표시하고 순차 탐색을 통해 빠진 수를 찾는다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int n = Integer.parseInt(br.readLine());
int p = Integer.parseInt(br.readLine());
boolean[] existing = new boolean[n + p];
StringTokenizer st = new StringTokenizer(br.readLine());
for (int i = 0; i < n - 1; i++) {
    existing[Integer.parseInt(st.nextToken())] = true;
}
for (int i = p; i < n + p; i++) {
    if (!existing[i]) {
        System.out.println(i);
    }
}
```
