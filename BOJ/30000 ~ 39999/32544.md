## 문제
$n$명의 사람들로 몇 층의 피라미드를 쌓을 수 있는지 구하는 문제

## 아이디어
Brute Force, Math

Naive 하게 구현해도 풀린다. $O(\sqrt{n}​)$
```python
n = int(input())
layer = 0
while True:
    if layer + 1 > n:
        break
    layer += 1
    n -= layer
print(layer)
```
