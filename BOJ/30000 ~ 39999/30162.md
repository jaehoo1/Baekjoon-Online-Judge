## 문제
$3 × n$ 타일을 ㄱ 자 모양의 블럭으로 채우는 방법의 수를 구하는 문제

![](https://upload.acmicpc.net/6c108111-7a33-4aaf-9ef6-79c1239c8f87/)

## 아이디어
Math

$3 × 2$ 타일을 채우는 경우는 2가지다. $3 × 2$ 타일이 $n / 2$ 개 있는 것 이므로 $2^{n/2}$
```java
int n = new Scanner(System.in).nextInt();
System.out.println(n % 2 == 0 ? (int) Math.pow(2, n / 2) : 0);
```
