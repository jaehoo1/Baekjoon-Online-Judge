# [33909 : 알파벳 블록](https://www.acmicpc.net/problem/33909)
https://www.acmicpc.net/problem/33909

## 아이디어
Arithmetic(Math), Implementation

S는 N으로 쓸 수 있고, O는 C 두 개로 사용할 수 있다.
```java
Scanner sc = new Scanner(System.in);
int s = sc.nextInt();
int c = sc.nextInt();
c += sc.nextInt() * 2;  // O
s += sc.nextInt();      // N
```

한 박스에는 SCON, SCCC 가 들어가야 하는데 N과 O를 S와 C로 변경하여 계산하면 된다.
```java
System.out.println(Math.min(s / 3, c / 6));
```
