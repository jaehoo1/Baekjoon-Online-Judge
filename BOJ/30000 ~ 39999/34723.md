# [34723 : 피막치](https://www.acmicpc.net/problem/34723)
https://www.acmicpc.net/problem/34723

## 아이디어
Brute Force, Arithmetic(Math)

피막치 공식을 가능한 $P$, $M$, $C$ 모든 경우의 수에 대해 계산해 최소값을 찾는다. (범위가 작아서 가능)
```java
Scanner sc = new Scanner(System.in);
int P = sc.nextInt();
int M = sc.nextInt();
int C = sc.nextInt();
int X = sc.nextInt();
int min = Integer.MAX_VALUE;
for (int p = 1; p <= P; p++) {
    for (int m = 1; m <= M; m++) {
        for (int c = 1; c <= C; c++) {
            min = Math.min(
                    Math.abs((p + m) * (m + c) - X),
                    min
            );
        }
    }
}
System.out.println(min);
```
