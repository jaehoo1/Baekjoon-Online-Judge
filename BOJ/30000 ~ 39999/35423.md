## 아이디어
String, Implementation

`'SciCom'`, `Love` 에서 `S` 와 `L`은 각 문자열에서 유일하다  
→ 뒤죽박죽인 문자열이 더라도 'S'와 'L' 이 몇 번 나오는 지를 세면 `'SciCom'`, `Love` 가 총 몇 번 나왔는지 알 수 있다.
```python
S = input()
print("SciCom" * S.count('S') + "Love" * S.count('L'))
```
