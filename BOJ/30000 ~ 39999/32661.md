# [32661 : Airfare Grants](https://www.acmicpc.net/problem/32661)
https://www.acmicpc.net/problem/32661

## 문제
**문제**  
축하합니다! 여러분의 팀이 국제 컵케이크 생산 대회(ICPC) 다음 라운드에 진출했습니다!

대회에 참가하려면 여러분은 고향에서 컵케이크 시티로 날아가야 합니다. 대회 주최 측에서 방금 좋은 소식을 전해왔습니다. 관대한 스폰서가 항공권 비용을 지원해드릴 예정입니다!

이제부터 어떻게 하는지 보여드릴겁니다.

여러분은 이미 이용 가능한 항공편 목록을 확보했습니다.

이 중에서 하나의 항공편 가격을 신고해야 합니다. 이는 여행의 환급 한도를 결정하며 스폰서는 그 가격의 최대 절반까지 여러분에게 환급해드립니다.

다음으로 실제로 항공권을 구매합니다. 이 항공편은 환불 한도에 대해 신고하는 항공편과 동일할 수도 있고 그렇지 않을 수도 있습니다.

마지막으로 구매 영수증을 보여주시면 스폰서가 여러분에게 실제 지불한 금액과 환급 한도 중 더 적은 금액을 환불해 드립니다.

이제 최소 가능한 순지불 비용, 즉 지불하는 가격에서 상환되는 금액을 뺀 금액을 계산해야 합니다.

당연히 가장 비싼 항공권의 가격을 신고하고 가장 저렴한 항공권을 구매하게 됩니다.

<br/>

**입력**  
첫 번째 입력 줄은 정수 $N$ ($1 \leq N \leq 50$)로, 이용 가능한 비행 횟수입니다.

다음 $N$개의 각 라인에는 고향에서 컵케이크 시티로 가는 항공권 가격인 정수 $P$ ($10 \leq P \leq 10^5$),가 달러로 표시되어 있습니다. $P$는 $10$의 배수임이 보장됩니다.

<br/>

**출력**  
스폰서의 여행 지원 금액을 제외하고 항공권에 지불해야 하는 최소 금액인 단일 정수를 달러로 출력합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/161627

## 아이디어
Arithmetic(Math)

항공편 가격의 최대, 최소값을 구한다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int N = Integer.parseInt(br.readLine());
int max = 0, min = Integer.MAX_VALUE;
while (N-- > 0) {
    int P = Integer.parseInt(br.readLine());
    max = Math.max(max, P);
    min = Math.min(min, P);
}
```

최대값 / 2 만큼은 지원을 해주므로 항공편 가격 최소값이 최대값 / 2 보다 큰 지를 비교하면 된다.
```java
System.out.println(min > max / 2 ? min - max / 2 : 0);
```
