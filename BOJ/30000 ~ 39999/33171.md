## 문제
**문제**  
양의 정수 N, A, B가 주어진다. 이때 A ≠ B이다.

1 이상 N 이하의 정수 중 A와 B 중 어느 한쪽으로만 나누어떨어지는 정수의 개수를 구하여라.

<br/>

**입력**  
입력은 다음과 같이 주어진다.
```text
N
A
B
```

<br/>

**출력**  
1 이상 N 이하의 정수 중 A와 B 중 어느 한쪽으로만 나누어떨어지는 정수의 개수를 단위(개)를 제외하고 출력한다.

답 이외에 아무것도 출력해서는 안 된다. (입력을 요구하는 문장 등도 출력하지 않는다.)

<br/>

번역 출처 : https://www.acmicpc.net/board/view/160550

## 아이디어
Brute Force, Arithmetic(Math)

N 범위가 작기 때문에 1 ~ N까지의 모든 수를 A와 B로 나누어 떨어지는 지 확인하면 된다.
```python
N = int(input())
A = int(input())
B = int(input())

count = 0
for i in range(1, N + 1):
    if (i % A == 0 and i % B != 0) or (i % A != 0 and i % B == 0):
        count += 1
print(count)
```
