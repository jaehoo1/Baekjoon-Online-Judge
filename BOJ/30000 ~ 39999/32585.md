# [32585 : Building Pyramids](https://www.acmicpc.net/problem/32585)
https://www.acmicpc.net/problem/32585

## 문제
**문제**  
마침내 고양이들은 코아티(너구리의 친척)와 벌들과의 다툼을 끝내기로 했습니다. 고양이들은 화해 기념하기 위해 기념비를 세우기로 결정합니다. 이렇게 다양한 동물들이 조화롭게 살 수 있다는 것을 보여주기 위해 위대한 건축가 코아티는 기념비가 매우 날카로운 모서리를 가진 삼각뿔과 모서리가 전혀 없는 구의 두 가지 완전히 다른 3차원 모양으로 구성되어야 한다고 결정합니다. 이 아이디어는 그림 B.1에 표시된 것과 유사하게 많은 작은 구체로 큰 정사면체를 만드는 것입니다.

![](https://upload.acmicpc.net/1fa91653-a93c-429a-ac21-e0a114ccfcdb/-/preview/)

그림 B.1: 두 번째 샘플 입력에 해당하는 구의 삼각형 피라미드입니다. 피라미드의 한쪽 가장자리에는 $5$개의 구가 있으므로 전체 피라미드는 $35$개의 구로 구성됩니다.

코아티 알리시아, 꿀벌 비즈워스, 고양이 다라벨라는 이 기념비가 가능한 한 커지기를 원합니다. 그들은 기념비를 놓을 땅을 지정하고 기념비를 놓을 위치를 기준으로 가장자리 중 하나에 몇 개의 구가 있어야 하는지 결정했습니다. 가장자리 중 하나에 있는 구의 수를 알면 피라미드 전체에 몇 개의 구가 포함될까요?

<br/>

**입력**  
입력은 다음과 같이 구성됩니다:
- 정수 $n(1 \le n \le 10^6)$을 가진 한 줄, 피라미드의 가장자리 중 하나에 있는 구의 수입니다.

<br/>

**출력**  
피라미드 전체에 있는 구의 수를 출력합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/161615

## 아이디어
Math

예를 들어 $n == 5$인 경우에 구의 개수는 다음과 같이 구할 수 있는데
```text
1층 : 1
2층 : 1 + 3(1 + 2)
3층 : 1 + 3(1 + 2) + 6(1 + 2 + 3)
4층 : 1 + 3(1 + 2) + 6(1 + 2 + 3) + 10(1 + 2 + 3 + 4)
5층 : 1 + 3(1 + 2) + 6(1 + 2 + 3) + 10(1 + 2 + 3 + 4) + 15(1 + 2 + 3 + 4 + 5)
```

이를 기반으로 $n$에 대한 공식을 구할 수 있다. 가우스(시그마) 공식을 써도 된다.
```java
int n = new Scanner(System.in).nextInt();
long spheres = 0;
for (int i = 1; i <= n; i++) {
    spheres += (long) i * (n - i + 1);
}
System.out.println(spheres);
```
