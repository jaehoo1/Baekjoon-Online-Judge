# [34459 : MIT Time](https://www.acmicpc.net/problem/34459)
https://www.acmicpc.net/problem/34459

## 문제
주어진 수 $N$이 해당하는 $5^{k-1} < N ≤ 5^k$ 범위 $k$ 찾는 문제

## 아이디어
Brute Force, Arithmetic(Math), Implementation

범위에 해당하는 $k$를 찾을 때 까지 $5^k$의 $k$를 증가시켜 찾는다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int T = Integer.parseInt(br.readLine());
while (T-- > 0) {
    int N = Integer.parseInt(br.readLine());
    // 범위에 해당하는 k 찾기
    int k;
    for (k = 1; Math.pow(5, k) < N; k++);
    bw.write("MIT" + (k > 1 ? "^" + k : "") + " time\n");
}
```
