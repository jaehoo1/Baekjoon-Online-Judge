# [32921 : False Alarm](https://www.acmicpc.net/problem/32921)
https://www.acmicpc.net/problem/32921

## 문제
10분 내 3번 알람이 울려야 (잠에서) 깰 수 있다. 깨기 위해 추가해야하는 알람의 최소 수를 구하는 문제

## 아이디어
Math, Implementation

계산/구현 간편화를 위해 시:분을 분으로 치환
```java
static int minTime(String time) {
    StringTokenizer st = new StringTokenizer(time, ":");
    return Integer.parseInt(st.nextToken()) * 60 + Integer.parseInt(st.nextToken());
}


BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int n = Integer.parseInt(br.readLine());
int[] times = new int[n];
for (int i = 0; i < n; i++) {
    times[i] = minTime(br.readLine());
}
```

- 10분 안에 3번의 알람이 울리는 경우가 있다면 알람을 추가하지 않아도 일어날 수 있음 (답: 0)
- 10분 안에 2번의 알람이 울리는 경우가 있다면 알람을 1개 추가하면 일어날 수 있음 (답: 1)
- 나머지 (10분 안에 1번의 알람이 울리는 경우) 경우에는 알람을 2개 추가하면 일어날 수 있음 (답: 2)
```java
for (int i = 0; i < n - 2; i++) {
    if (times[i + 2] - times[i] <= 10) {
        System.out.println(0);
        System.exit(0);
    }
}
for (int i = 0; i < n - 1; i++) {
    if (times[i + 1] - times[i] <= 10) {
        System.out.println(1);
        System.exit(0);
    }
}
System.out.println(2);
```
