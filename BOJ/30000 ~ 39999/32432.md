# [32432 : Шифровка](https://www.acmicpc.net/problem/32432)
https://www.acmicpc.net/problem/32432

## 문제
**문제**  
비니시우스는 “강연 및 논평 컨설팅 기관(ICPC)”의 이사회 회의에 참석하고 있습니다. 그는 이사들이 발언 시간을 좀 더 간결하게 지켜서 회의가 점심시간 전에 끝났으면 좋겠다고 생각하고 있습니다. 안타깝게도 기관의 특성 때문인지, 모두들 말하는 것을 무척 좋아합니다.

회의에서 연설할 $N$명의 이사들이 있으며, 각 이사들은 동일한 시간 동안 연설할 것입니다. 두 연속된 연설 사이에는 1분 간격이 있습니다, 회의가 $K$분을 넘지 않도록 각 연설의 최대 시간을 분 단위로 구하세요.

<br/>

**입력**  
첫 번째 줄에는 정수 $N(1 ≤ N ≤ 100)$, 즉 이사 수가 포함되어 있습니다. 두 번째 줄에는 정수 $K(1 ≤ K ≤ 1000$ 및 $K ≥ N)$가 포함되어 있으며, 이는 최대 회의 시간(분 단위)입니다. 모든 입력 경우에 대해 각 이사의 연설은 최소 1분 동안 지속됩니다.

<br/>

**출력**  
프로그램은 각 이사회 멤버의 연설 길이를 나타내는 단일 정수가 포함된 하나의 정수를 분 단위로 출력해야 합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/161563

## 아이디어
Arithmetic(Math), Brute Force, Implementation

공식을 도출해 답을 구해도 되지만, $N$, $K$ 범위가 작아 경우의 수 탐색(Brute Force)으로도 구현이 가능하다.
```java
Scanner sc = new Scanner(System.in);
int N = sc.nextInt();
int K = sc.nextInt();
int i;
for (i = 1; N * i + N - 1 <= K; i++);   // N명의 연설 + (N - 1)번의 쉬는 시간이 K가 넘지 않는 최대값 찾기
System.out.println(i - 1);
```
