# [32800 : Bus Assignment](https://www.acmicpc.net/problem/32800)
https://www.acmicpc.net/problem/32800

## 문제
**문제**  
사람을 조심히 태우는 기관은 Line Town의 유명한 라인 버스를 관리합니다. 라인 버스는 $1$번부터 $n$번까지 번호가 매겨진 정류장을 순서대로 지나갑니다. 정류장 $i$에서, 먼저 $aᵢ$명이 버스에서 내리고, 그 다음 $bᵢ$명이 버스에 탑니다. 버스는 $1$번 정류장에서 비어있는 상태로 시작하며, 정류장을 번호 순서대로 지나갑니다. 마지막인 $n$번 정류장에서 모든 승객이 하차하여 버스가 완전히 비어야 합니다.

라인 버스를 이용하는 승객들은 모두 반드시 자리에 앉아야 합니다. 버스의 수용 인원은 $c$명이면 정확히 $c$개의 좌석이 있는 것으로 간주합니다. 운전자는 좌석 수에 포함되지 않습니다. 사람을 조심히 태우는 기관은 라인 버스를 운행하기 위해 필요한 최소 버스 좌석 수(최소 수용 인원)를 알고 싶어합니다.

<br/>

**입력**  
첫 번째 줄에 정수 $n$이 주어집니다. $(2 ≤ n ≤ 2 \cdot 10⁵)$ 

그 다음 $n$줄에 걸쳐 각각 정수 $aᵢ$, $bᵢ$가 주어집니다. $(0 ≤ aᵢ, bᵢ ≤ 10⁹)$

적어도 한 명 이상은 반드시 버스에 탑니다. 전체 정류장을 통틀어 버스에 탄 사람 수의 합은 최대 $10⁹$명 이하입니다. $n$번 정류장에서 버스는 반드시 빈 상태가 되어야 합니다.

<br/>

**출력**  
필요한 최소 버스 좌석 수를 하나의 정수로 출력하세요.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/159214

## 아이디어
Arithmetic(Math), Implementation

버스에서 정류장마다 타고 내리는 사람에 따라 사람 수에 뺄셈, 덧셈을 하고 그 때마다 최대값을 확인한다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int busPeople = 0;
int max = 0;
int n = Integer.parseInt(br.readLine());
while (n-- > 0) {
    StringTokenizer st = new StringTokenizer(br.readLine());
    int a = Integer.parseInt(st.nextToken());   // 정류장에서 내리는 사람 수
    int b = Integer.parseInt(st.nextToken());   // 정류장에서 타는 사람 수
    busPeople += -a + b;                        // 정류장마다 갱신된 사람 수
    max = Math.max(max, busPeople);             // 최대값 확인
}
System.out.println(max);
```
