# [32500 : Dishonest Lottery](https://www.acmicpc.net/problem/32500)
https://www.acmicpc.net/problem/32500

## 문제
복권 1회 추첨 할 때 마다 5 * 10개의 무작위 선택되는 수들이 나온다.  
n 회 추첨되는 복권 표본이 주어질 때 자주 나오는 수(2 * n 회 초과)를 찾는 문제

## 아이디어
Implementation

수가 몇 번 나왔는지 세는 배열을 만들어 나온 횟수를 센다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
int[] counts = new int[51];
int n = Integer.parseInt(br.readLine());
boolean output = false;
for (int i = 0; i < n; i++) {
    for (int j = 0; j < 10; j++) {
        StringTokenizer st = new StringTokenizer(br.readLine());
        for (int k = 0; k < 5; k++) {
            counts[Integer.parseInt(st.nextToken())]++;
        }
    }
}
```

2n 번 초과인 횟수를 찾는다. 모두 균등하게 나왔을 경우에는 -1을 출력해 줘야함
```java
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

for (int i = 1; i <= 50; i++) {
    if (counts[i] > 2 * n) {
        bw.write(i + " ");
        output = true;
    }
}
if (!output) {
    bw.write("-1");
}
```
