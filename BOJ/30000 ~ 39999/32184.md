## 아이디어
Simulation(Implementation), Brute Force

$A$, $B$ 범위가 작아 책을 넘기는(사진 찍는) 것을 직접 구현해봐도 된다.

$A$ 페이지부터 시작해서 짝수 페이지면 1장, 홀수 페이지면 2장씩 찍으면 된다.
```java
Scanner sc = new Scanner(System.in);
int A = sc.nextInt();
int B = sc.nextInt();
int picture = 0;
while (A <= B) {
    picture++;
    A += A % 2 == 0 ? 1 : 2;
}
System.out.println(picture);
```
