## 문제
**문제**  
소수를 튜플로 생각하고 (대소)비교했을 때 원본 소수와의 비교 결과가 같은지 구하는 문제

## 아이디어
Parsing(String), Implementation

소수를 정수, 소숫점 부분으로 나눠 대소 비교의 결과가 원래 수와의 대소 비교와 같은지 판별한다.
```python
a = input()
a_float = float(a)  # 원본 소수
a_int, a_frac = map(int, a.split('.'))  # 정수부, 소수점부
b = input()
b_float = float(b)  # 원본 소수
b_int, b_frac = map(int, b.split('.'))  # 정수부, 소수점부

cmp = (a_float > b_float) - (a_float < b_float)     # 원본 수 대소비교
int_cmp = (a_int > b_int) - (a_int < b_int)         # 정수부 대소비교
frac_cmp = (a_frac > b_frac) - (a_frac < b_frac)    # 소수점부 대소비교

if int_cmp != 0:    # 정수부가 다르면 결과가 명확함 (정수가 큰 부분이 큰 수)
    print(max(a_float, b_float))
else:               # 정수부가 같다면 원본수와 소수부의 대소비교가 같은지 확인
    print(max(a_float, b_float) if cmp == frac_cmp else -1)
```
