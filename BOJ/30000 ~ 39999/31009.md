# [31009 : 진주로 가자! (Easy)](https://www.acmicpc.net/problem/31009)
https://www.acmicpc.net/problem/31009

## 아이디어
String, Implementation

진주가 어디서 나왔는지 찾는다  
(진주가 나온 인덱스를 저장)
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
int N = Integer.parseInt(br.readLine());
String[] D = new String[N];
int[] C = new int[N];
int jinju = -1;
for (int i = 0; i < N; i++) {
    StringTokenizer st = new StringTokenizer(br.readLine());
    D[i] = st.nextToken();
    C[i] = Integer.parseInt(st.nextToken());
    if (D[i].equals("jinju")) {
        jinju = i;
    }
}
```

진주가 나온 인덱스를 알고 있으므로, 진주행의 가격과 진주행보다 비싼 것을 찾을 수 있다.  
교통편 리스트를 재순차탐색하여 진주행보다 비싼 것의 개수를 센다.
```java
int count = 0;

for (int i = 0; i < N; i++) {
    if (C[i] > C[jinju]) {
        count++;
    }
}
System.out.println(C[jinju]);
System.out.println(count);
```
