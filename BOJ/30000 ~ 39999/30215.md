# [30215 : Age Expression](https://www.acmicpc.net/problem/30215)
https://www.acmicpc.net/problem/30215

## 문제
Dr.O, Alyssa, Konari의 나이가 주어질 때 Dr.O’s age == (a × Alyssa’s age) + (k × Konari’s age) 인 a와 k가 있는지 찾는 문제

## 아이디어
Brute Force, Arithmetic(Math)

이중 `for` 문으로 가능한 모든 a, k를 (하나라도) 찾는다.
```java
Scanner sc = new Scanner(System.in);
int drO = sc.nextInt();
int alyssa = sc.nextInt();
int konari = sc.nextInt();
for (int a = 1; a * alyssa < drO; a++) {
    for (int k = 1; a * alyssa + k * konari <= drO; k++) {
        if (a * alyssa + k * konari == drO) {
            System.out.println(1);
            System.exit(0);
        }
    }
}
System.out.println(0);
```
