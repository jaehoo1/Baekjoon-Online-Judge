# [31312 : Water Journal](https://www.acmicpc.net/problem/31312)
https://www.acmicpc.net/problem/31312

## 문제
**문제**  
유진은 매일 자신이 마신 물의 양(컵)을 기록합니다. 하지만 그는 일지에 하루치 기록을 잊어버렸습니다. 다행히도 그는 일지를 작성하기 시작한 이후로 매일 마신 물의 최소량과 최대량을 기억하고 있으며, 최소량을 마신 날이 최소 하루, 최대량을 마신 날도 최소 하루는 있었다는 것을 기억합니다. 그가 기록을 잊어버린 날 마셨을 수 있는 모든 가능한 물의 양을 계산해주세요.

<br/>

**입력**  
입력의 첫 번째 줄에는 세 개의 정수 $n$ $(2 \le n \le 50)$, $a$와 $b$ $(1 \le a \le b \le 50)$가 주어집니다. 여기서 $n$은 유진의 일지에 있는 날짜 수이고, $a$는 하루 동안 마신 물의 최소 컵 수, $b$는 최대 컵 수입니다.

다음 $n-1$개의 줄에는 각각 하나의 정수 $w$ $(a \le w \le b)$가 주어집니다. 이는 유진이 기록을 잊어버린 날을 제외한 모든 날의 일지 기록입니다.

<br/>

**출력**  
누락된 기록에 가능한 모든 값을 오름차순으로 공백으로 구분하여 출력하세요. 만약 유진의 기억이 잘못되어 가능한 값이 없다면 -1을 출력하세요.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/152401

## 아이디어
Case Work(Implementation)

$a$(최소값), $b$(최대값) 등장 여부에 따라 분기 처리하면 된다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
StringTokenizer st = new StringTokenizer(br.readLine());
int n = Integer.parseInt(st.nextToken());
int a = Integer.parseInt(st.nextToken());
int b = Integer.parseInt(st.nextToken());
boolean min = false, max = false;
for (int i = 0; i < n - 1; i++) {
    int w = Integer.parseInt(br.readLine());
    // 최대 / 최소값이 나왔는지 기록
    if (w == a) {
        min = true;
    }
    if (w == b) {
        max = true;
    }
}
if (!min && !max) {     // 최대, 최소값이 둘 다 나온 적 없다면 불가능(기억이)
    System.out.println(-1);
} else if (!min) {      // 최소값이 나온 적 없다면 최소값만 가능
    System.out.println(a);
} else if (!max) {      // 최대값이 나온 적 없다면 최대값만 가능
    System.out.println(b);
} else {                // 최대, 최소값이 둘 다 나왔다면 범위 내 모든 수 가능
    for (int i = a; i <= b; i++) {
        System.out.println(i);
    }
}
```
