# [33753 : 주차 요금 정산하기](https://www.acmicpc.net/problem/33753)
https://www.acmicpc.net/problem/33753

## 아이디어
Arithmetic(Math)

입력 처리와 요금 초기화
```python
A, B, C = map(int, input().split())
T = int(input())
fee = 0
```

기본 요금 $A$원 부과 (하면서 시간 차감)
```python
T -= 30
fee += A
if T < 0:
    T = 0
```

나머지 시간에 대해서도 요금 부과
```python
# B분으로 안나누어 떨어지면 올림
if T % B != 0:
    fee += C
# B분마다 C원
fee += T // B * C

print (fee)
```
