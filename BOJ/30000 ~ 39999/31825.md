# [31825 : 알파벳과 쿼리 (Easy)](https://www.acmicpc.net/problem/31825)
https://www.acmicpc.net/problem/31825

## 아이디어
String, Implementation

$N$, $Q$ 제한이 작으므로 주어지는 쿼리를 naive하게 구현하면 된다.

문자열 중간을 바꾸는 연산을 해야하므로, `char[]`로 처리
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

StringTokenizer st = new StringTokenizer(br.readLine());
st.nextToken();     // N 필요 없음
int Q = Integer.parseInt(st.nextToken());
char[] S = br.readLine().toCharArray();
```

쿼리 1, 2 구현
```java
static int query1(char[] chars, int l, int r) {
    l--;
    r--;
    int cnt = 1;
    for (int i = l + 1; i <= r; i++) {
        if (chars[i] != chars[i - 1]) {
            cnt++;
        }
    }
    return cnt;
}

static void query2(char[] chars, int l, int r) {
    l--;
    r--;
    while (l <= r) {
        chars[l]++;
        if (chars[l] > 'Z') {
            chars[l] = 'A';
        }
        l++;
    }
}
```

쿼리 처리 구현
```java
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

while (Q-- > 0) {
    st = new StringTokenizer(br.readLine());
    int cmd = Integer.parseInt(st.nextToken());
    int l = Integer.parseInt(st.nextToken());
    int r = Integer.parseInt(st.nextToken());
    if (cmd == 1) {
        bw.write(query1(S, l, r) + "\n");
    } else if (cmd == 2) {
        query2(S, l, r);
    }
}
```
