# [34454 : Roller Coaster Ride](https://www.acmicpc.net/problem/34454)
https://www.acmicpc.net/problem/34454

## 문제
**문제**  
당신은 CEMC의 놀이공원에 하루를 보내고 있습니다. 놀이공원에는 한 대의 열차가 있는 롤러코스터가 있습니다. 열차는 여러 대의 객차(car)로 이루어져 있고, 각 객차는 동일한 수의 사람들을 태울 수 있습니다.

당신이 롤러코스터에 도착했을 때, 이미 줄이 있습니다. 당신의 임무는, 각 열차가 항상 꽉 차서 출발한다고 가정했을 때, 당신이 다음 열차에 탈 수 있는지 여부를 결정하는 것입니다.

<br/>

**입력**  
첫 번째 줄: 정수 $N$ → 줄에서의 당신의 순번 (예: $N = 5$이면 당신은 5번째 사람)

두 번째 줄: 정수 $C$ → 열차의 객차 수

세 번째 줄: 정수 $P$ → 한 객차에 탑승할 수 있는 인원 수

<br/>

**출력**  
yes 또는 no를 출력합니다. (당신이 다음 열차에 탈 수 있으면 yes, 아니면 no)

<br/>

번역 출처 : https://www.acmicpc.net/board/view/162869

## 아이디어
Arithmetic(Math)

내가 서있는 순번($N$)이 한 번에 탈수 있는지($C * P$)를 비교하면 된다.
```python
N = int(input())
C = int(input())
P = int(input())
print("yes" if N <= C * P else "no")
```
