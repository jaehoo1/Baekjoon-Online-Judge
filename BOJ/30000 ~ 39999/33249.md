## 문제
원기둥 기둥넓이 + 바닥(원) 넓이를 구하는 문제

## 아이디어
Geometry(Math)

반지름을 구해 원 넓이와 기둥 넓이를 구한다.
```python
import math

d, h = map(float, input().split())
r = d / 2 + 5
print(math.pi * r * r + 2 * math.pi * r * h)
```
