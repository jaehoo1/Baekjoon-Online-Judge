## 아이디어
String, Brute Force

`eagle` 이랑 몇 글자가 겹치는지 확인을 해야한다.
```java
static final String EAGLE = "eagle";

/** s가 index에서 시작해서 eagle과 몇 개의 문자가 일치하는가? */
static int matches(String s, int index) {
    if (index + 5 > s.length()) {
        return 0;
    }
    int cnt = 0;
    for (int i = 0; i < 5; i++) {
        if (s.charAt(i + index) == EAGLE.charAt(i)) {
            cnt++;
        }
    }
    return cnt;
}
```

모든 문자열 인덱스에서 `eagle`과 가장 많이 겹치는 횟수를 찾으면 최소한의 글자를 바꿔 `eagle`을 만들 수 있다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int N = Integer.parseInt(br.readLine());
String s = br.readLine();
int max = 0;
for (int i = 0; i < N; i++) {
    max = Math.max(max, matches(s, i));
}
System.out.println(5 - max);
```
