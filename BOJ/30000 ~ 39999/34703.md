# [34703 : 공강 사수](https://www.acmicpc.net/problem/34703)
https://www.acmicpc.net/problem/34703

## 아이디어
Implementation

월~금에 강의가 있는지의 여부를 강의 입력 마다 `boolean[]`에 갱신해 공강이 있나를 확인한다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
boolean[] lectures = new boolean[6];
int N = Integer.parseInt(br.readLine());
for (int i = 0; i < N; i++) {
    lectures[Integer.parseInt(br.readLine())] = true;
}
for (int i = 1; i < 6; i++) {
    if (!lectures[i]) {
        System.out.println("YES");
        System.exit(0);
    }
}
System.out.println("NO");
```
