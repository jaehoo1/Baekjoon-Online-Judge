# [31663 : Mean Words](https://www.acmicpc.net/problem/31663)
https://www.acmicpc.net/problem/31663

## 문제
문자들의 위치별 평균(소수점 있으면 내림)을 구하는 문제

## 아이디어
Arithmetic(Math), String, Implementation

평균을 구하기 위해 문자들의 위치별 합과 제수를 구해 놓는다
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
int[] sums = new int[45];
int[] lengths = new int[45];
int N = Integer.parseInt(br.readLine());
for (int i = 0; i < N; i++) {
    String word = br.readLine();
    for (int j = 0; j < word.length(); j++) {
        sums[j] += word.charAt(j);      // 아스키 값 합산
        lengths[j]++;                   // 제수 증가
    }
}
```

문자 위치별 평균 구하기
```java
for (int i = 0; i < 45 && lengths[i] > 0; i++) {
    System.out.print((char) Math.floor((double) sums[i] / lengths[i]));
}
```
