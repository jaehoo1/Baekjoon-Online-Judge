# [34437 : Number Reduction](https://www.acmicpc.net/problem/34437)
https://www.acmicpc.net/problem/34437

## 문제
$N$을 다음 연산을 통해 1로 만드려고 할 때 몇 번의 연산이 필요한지 구하는 문제
- $N$이 홀수면 $N$에 두 배 + 1을 더함
- $N$이 짝수면 2로 나눔

## 아이디어
Arithmetic(Math), Simulation(Implementation)

문제 조건을 그대로 구현(시뮬레이션)하면 된다.
```java
int count = 0;
int N = new Scanner(System.in).nextInt();
while (N != 1) {
    if (N % 2 == 0) {
        N /= 2;
    } else {
        N += N * 2 + 1;
    }
    count++;
}
System.out.println(count);
```
