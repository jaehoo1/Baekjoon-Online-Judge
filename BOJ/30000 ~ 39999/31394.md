# [31394 : Scholarship](https://www.acmicpc.net/problem/31394)
https://www.acmicpc.net/problem/31394

## 문제
**문제**  
드디어 대학교의 한 학기가 마무리 되었습니다. Vasya는 이미 모든 시험을 치뤘고 점수를 확인하였으며, 자신이 어떤 장학금을 받을 수 있는지 알고 싶어합니다. 본교의 장학금 지급 제도는 아래와 같습니다.
- 만약 학생이 한 과목이라도 '미'를 받는다면, 장학금은 지급되지 않습니다.
- 학생이 모든 과목에서 '수'를 받는다면 개인 장학금이 지급됩니다.
- 개인 장학금을 받지 못한 학생의 평균 점수가 4.5점 이상이라면 우수 장학금이 지급됩니다.
- 그 외의 경우에는 모두 일반 장학금이 지급됩니다.

이때, '수'는 5점에 해당하고, '우'는 4점에 해당하며, '미'는 3점에 해당합니다. 학생의 평균 점수는 모든 과목의 점수의 평균값입니다. Vasya가 어떤 장학금을 받게 되는지 알아내도록 도와주세요.

<br/>

**입력**  
첫째 줄에 전체 과목 수 $n$이 주어집니다. $(1 ≤ n ≤ 10)$

이후 $n$개의 줄에 걸쳐, 정수 $m_i$이 주어집니다. 이는 어느 특정 과목에서 Vasya가 받은 점수를 의미합니다. $(3 ≤ m_i ≤ 5)$

<br/>

**출력**  
Vasya가 어떤 장학금도 받지 않는다면 `None`, 일반 장학금을 받는다면 `Common`, 우수 장학금을 받는다면 `High`, 개인 장학금을 받는다면 `Named` 를 출력합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/135524

## 아이디어
Arithmetic(Math), Implementation

장학금 조건에 따른 분기처리 구현
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

// 시험 점수들 배열에 담고
int n = Integer.parseInt(br.readLine());
int[] marks = new int[n];
for (int i = 0; i < n; i++) {
    marks[i] = Integer.parseInt(br.readLine());
}

String result;
// 조건 분기 구현
if (Arrays.stream(marks).anyMatch(i -> i == 3)) {                   // 미가 존재한다면 장학금 없음
    result = "None";
} else if (Arrays.stream(marks).allMatch(i -> i == 5)) {            // 모두 수면 개인 장학금
    result = "Named";
} else if (Arrays.stream(marks).average().getAsDouble() >= 4.5) {   // 평균이 4.5 이상이면 우수 장학금
    result = "High";
} else {                                                            // 그 외는 일반 장학금
    result = "Common";
}
System.out.println(result);
```
