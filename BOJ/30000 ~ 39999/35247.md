## 문제
$n$을 저장하기 위해 필요한 최소 비트 수를 구하는 문제

## 아이디어
Arithmetic(Math)

비트 수 ^ 2 - 1 이 비트로 표현할 수 있는 최대값이니, $n$과 비교하며 2씩 곱해주면 된다.
```python
n = int(input())
bits = 1
while n > 2 ** bits - 1:
    bits *= 2
print(f'{bits} {"bit" if bits == 1 else "bits"}')
```
