# [33639 : Candy Store](https://www.acmicpc.net/problem/33639)
https://www.acmicpc.net/problem/33639

## 문제
**문제**  
사람들의 이름이 주어질 때 이니셜을 호출하면 1인을 특정하는게 가능한지 구하는 문제

## 아이디어
Data Structure(Map), String, Implementation

`key: initial, value: List<Full Name>` 형태로 사람들 이름을 저장한다.
```java
static String initial(String firstName, String lastName) {
    return firstName.substring(0, 1) + lastName.substring(0, 1);
}

static String initial(String name) {
    StringTokenizer st = new StringTokenizer(name);
    return initial(st.nextToken(), st.nextToken());
}


BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

Map<String, List<String>> initials = new HashMap<>();
StringTokenizer st = new StringTokenizer(br.readLine());
int N = Integer.parseInt(st.nextToken());
int Q = Integer.parseInt(st.nextToken());
while (N-- > 0) {
    String name = br.readLine();
    String initial = initial(name);
    initials.computeIfAbsent(initial, k -> new ArrayList<>())
            .add(name);
}
```

이제 주어지는 이니셜로 사람을 찾는다.
```java
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

while (Q-- > 0) {
    List<String> result = initials.get(br.readLine());
    if (result == null) {                   // 해당 이니셜이 한 명도 없음
        bw.write("nobody\n");
    } else if (result.size() > 1) {         // 해당 이니셜이 여러 명임
        bw.write("ambiguous\n");
    } else {                                // 유일한 경우
        bw.write(result.get(0) + "\n");
    }
}
```
