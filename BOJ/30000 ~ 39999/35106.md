# [35106 : 릴레이 가위바위보 게임](https://www.acmicpc.net/problem/35106)
https://www.acmicpc.net/problem/35106

## 아이디어
Implementation

가위바위보(1,2,3)가 몇 번 나왔는지 세어주고
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int N = Integer.parseInt(br.readLine());
Map<Integer, Integer> map = new HashMap<>();
IntStream.rangeClosed(1, 3)
        .forEach(i -> map.put(i, 0));
StringTokenizer st = new StringTokenizer(br.readLine());
for (int i = 0; i < N * 3; i++) {
    map.compute(Integer.valueOf(st.nextToken()), (k, v) -> v == null ? 1 : v + 1);
}
```

횟수가 $N - 1$(오류가 발생한 라운드에서 원래 출력되었어야 할 손동작의 번호),  
횟수가 $N + 1$(그 라운드에서 실제로 출력된 손동작의 번호) 인것을 찾는다.
```java
System.out.println(map.entrySet().stream().filter(entry -> entry.getValue() == N - 1).findFirst().get().getKey());
System.out.println(map.entrySet().stream().filter(entry -> entry.getValue() == N + 1).findFirst().get().getKey());
```
