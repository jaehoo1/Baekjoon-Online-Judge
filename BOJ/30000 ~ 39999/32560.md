# [32560 : Amalgram](https://www.acmicpc.net/problem/32560)
https://www.acmicpc.net/problem/32560

## 문제
**문제**  
**애너그램**은 알파벳의 각 글자가 원본과 정확히 같은 횟수로 나타나는 단어의 글자 배열입니다. 예를 들어, `clarinets`은 `larcenist`의 애너그램입니다.

**애멀그램**은 알파벳의 각 글자가 **두** 단어의 글자 중 하나의 원본 글자만큼 **최소** 여러 번 나타나는 배열입니다. 

예를 들어, `administration`는 `mantis`와 `raisin`의 애멀그램이지만, 두 단어 모두에 문자 `d`가 나타나지 않기 때문에 가능한 짧은 애멀그램이지는 않습니다.

두 단어가 주어지면 가능한 한 적은 글자를 포함하는 애멀그램을 출력하세요.

<br/>

**입력**  
- 단어 $a$를 나타내는 소문자 라틴 문자가 포함된 한 줄 $(1 \le |a| \le 10^6)$.
- 단어 $b$를 나타내는 소문자 라틴 문자가 포함된 한 줄 $(1 \le |b| \le 10^6)$.

<br/>

**출력**  
$a$와 $b$의 아말그램을 나타내는 최소 길이의 문자 시퀀스를 출력합니다. 답변이 여러 개인 경우 그 중 하나를 출력할 수 있습니다. 답변에 적어도 $a$의 모든 문자와 $b$의 모든 문자가 포함되어 있고 더 짧을 수 있는 다른 가능한 답변이 없는 경우 정답으로 판단됩니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/161610

## 아이디어
String, Implementation

$a$, $b$에 있는 문자 중 아무거나 골라서 쓰고 사용한 문자는 두 문자열에서 제거해주는 것을 $a$, $b$에 남은 문자가 없을 때 까지 해주면 된다.

$a$, $b$ 문자열에 있는 문자 카운팅
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int[] a = new int[26];
int[] b = new int[26];
for (char c : br.readLine().toCharArray()) {
    a[c - 'a']++;
}
for (char c : br.readLine().toCharArray()) {
    b[c - 'a']++;
}
```

$a$, $b$에 남아있는 문자가 없을 때 까지 문자를 꺼내서 추가해주면 된다.
```java
/**
 * 문자 카운팅 배열에서 남아있는 문자 찾기
 * 
 * @param arr 문자 카운팅 배열
 * @return 남아있는 문자 있다면 0 이상의 수(인덱스), 없다면 -1
 */
static int find(int[] arr) {
    for (int i = 0; i < 26; i++) {
        if (arr[i] > 0) {
            return i;
        }
    }
    return -1;
}

/**
 * 문자열에서 문자를 꺼냄
 * 
 * @param sb 꺼낸 문자를 붙이는 StringBuilder
 * @param idx 사용하는 (알파벳) 문자 index (0 ~ 25)
 * @param a 카운팅 배열 1
 * @param b 카운팅 배열 2
 */
static void addLetter(StringBuilder sb, int idx, int[] a, int[] b) {
    if (a[idx] > 0) {
        a[idx]--;
    }
    if (b[idx] > 0) {
        b[idx]--;
    }
    sb.append((char) ('a' + idx));
}


BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
StringBuilder sb = new StringBuilder();

while (true) {
    // a에 아직 문자가 남아있다면 (아직 안꺼낸 문자가 있다면) a에 있는 문자를 꺼냄
    int idx = find(a);
    if (idx == -1) {
        // a에 남은 문자가 없다면 b것을 사용
        idx = find(b);
    }
    // b도 남은 문자가 없으면 끝남
    if (idx == -1) {
        break;
    }
    addLetter(sb, idx, a, b);
}
bw.write(sb.toString());
```
