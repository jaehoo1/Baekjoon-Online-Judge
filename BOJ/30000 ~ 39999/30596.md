## 문제
막대 4개의 길이가 주어질 때 만들 수 있는 직사각형의 최대 넓이 구하기

## 아이디어
Geometry(Math)(, Brute Force)

맨 처음에는 Brute Force로 구현을 했었다 (물론 틀린 풀이는 아님)
```java
static int[] a = new int[4];
static boolean[] visited = new boolean[4];

static int max;

static int area(int[] sticks) {
    return Math.min(sticks[0], sticks[1]) * Math.min(sticks[2], sticks[3]);
}

static void dfs(int[] tree, int depth) {
    if (depth == 4) {
        max = Math.max(max, area(tree));
        return;
    }
    for (int i = 0; i < 4; i++) {
        if (visited[i]) {
            continue;
        }
        visited[i] = true;
        tree[depth] = a[i];
        dfs(tree, depth + 1);
        visited[i] = false;
    }
}


BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

for (int i = 0; i < 4; i++) {
    a[i] = Integer.parseInt(br.readLine());
}
dfs(new int[4], 0);
System.out.println(max);
```
