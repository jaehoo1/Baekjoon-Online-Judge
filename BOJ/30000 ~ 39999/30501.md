## 아이디어
String, Implementation

문제 그대로 `S`가 들어간 문자열을 찾으면 된다.
```python
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

int N = Integer.parseInt(br.readLine());
while (N-- > 0) {
    String name = br.readLine();
    if (name.contains("S")) {
        System.out.println(name);
    }
}
```
