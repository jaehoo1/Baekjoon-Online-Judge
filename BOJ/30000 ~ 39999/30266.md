# [30266 : Hurricane Warning](https://www.acmicpc.net/problem/30266)
https://www.acmicpc.net/problem/30266

## 문제
**문제**  
허리케인이 다가올 때, 허리케인 경로 상이나 근처에 있는 가능한 많은 사람들에게 경고하는 것이 매우 중요합니다. 이를 통해 사람들은 이상적으로 더 안전한 곳으로 이동하거나, 집을 보강하고, 만약 머물러야 한다면 필수품을 비축하고 백업 전원 시스템을 설치하는 등의 조치를 취할 수 있습니다. 문제는 어떻게 가능한 많은 사람들에게 도달할 수 있는가입니다. TV로 공지를 한다면, TV를 보지 않는 사람들은 경고와 지시사항을 놓칠 수 있습니다. 여기서는 다른 사람들이 소비하는 미디어와 당국이 경고에 사용하는 미디어에 대한 설명이 주어지며, 몇 명의 사람들이 경고를 받았는지 결정해야 합니다.

구체적으로, 이 정보는 문자열로 인코딩됩니다. 각 문자는 하나의 미디어 유형에 해당합니다. 예를 들어, 'T'는 TV, 'F'는 페이스북, 'X'는 전 트위터를 나타냅니다. 사용된 미디어 문자열과 소비된 미디어 문자열 사이에 공통 문자가 있다면 그 사람은 정보를 받게 됩니다. 그렇지 않으면 받지 못합니다.

<br/>

**입력**  
첫 번째 줄에는 1 ≤ K ≤ 100인 숫자 K가 주어집니다. 이는 파일에 있는 입력 데이터 세트의 수입니다. 이어서 K개의 데이터 세트가 다음 형식으로 주어집니다:

데이터 세트의 첫 번째 줄에는 0 ≤ n ≤ 10000인 정수 n이 주어집니다. 이는 고려하는 사람의 수입니다.

다음 줄은 1-10개의 대문자로 구성된 문자열로, 허리케인을 알리는 데 사용된 미디어를 설명합니다. 그 다음에 n개의 줄이 이어지며, 각 줄은 1-10개의 대문자로 구성됩니다. i번째 줄은 i번째 사람이 소비하는 미디어를 나타냅니다.

<br/>

**출력**  
각 데이터 세트에 대해, 먼저 "Data Set x:"를 별도의 줄에 출력하세요. 여기서 x는 데이터 세트의 번호입니다. 그 다음 허리케인 경고를 받은 사람의 수(최소 한 가지 유형의 미디어를 통해)를 출력하세요.

각 데이터 세트 뒤에는 빈 줄이 있어야 합니다.

<br/>

번역 출처 : https://www.acmicpc.net/board/view/158061

## 아이디어
Brute Force, String, Implementation

사람들이 소비한 미디어가 경고에 사용된 미디어인지를 확인해야 한다.
```java
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

int K = Integer.parseInt(br.readLine());
for (int k = 1; k <= K; k++) {
    int n = Integer.parseInt(br.readLine());
    // 경고에 사용된 미디어 초기화 (중복제거, 순서 의미 없음 → Set)
    Set<Character> announce = new HashSet<>();
    for (char c : br.readLine().toCharArray()) {
        announce.add(c);
    }

    int received = 0;
    while (n-- > 0) {
        // 사람들마다 소비한 미디어
        String consumedMedias = br.readLine();
        // 사람들이 소비한 미디어가 경고에 사용된 미디어인지 확인
        for (char c : consumedMedias.toCharArray()) {
            if (announce.contains(c)) {
                received++;
                break;
            }
        }
    }
    bw.write("Data Set " + k + ":\n");
    bw.write(received + "\n\n");
}
```
